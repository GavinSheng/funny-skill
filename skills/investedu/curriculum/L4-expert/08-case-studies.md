## 经典成功案例

### 案例 1: 巴菲特投资可口可乐 (1988)

#### 背景
- **时间**: 1987 年股灾后
- **市场情绪**: 极度悲观
- **公司状况**: 品牌强大但短期受挫

#### 投资逻辑
| 因素 | 分析 |
|------|------|
| 护城河 | 全球最强品牌之一 |
| 管理层 | 郭思达领导，优秀资本配置 |
| 财务 | 高 ROE(30%+)，稳定现金流 |
| 估值 | PE 约 15 倍，合理 |
| 成长 | 全球扩张空间大 |

#### 投资过程
- **买入**: 1988-1989 年，约 13 亿美元
- **持仓**: 持有至今，未减持
- **回报**: 成本 13 亿，现值约 250 亿+，分红超 200 亿

#### 关键启示
1. 好公司 + 好价格 = 好投资
2. 长期持有优质资产
3. 品牌护城河的价值
4. 逆向投资的机会

### 案例 2: 巴菲特投资苹果 (2016)

#### 背景
- **时间**: 2016 年
- **市场担忧**: iPhone 增长见顶
- **巴菲特转变**: 从"不投科技"到重仓

#### 投资逻辑
| 因素 | 分析 |
|------|------|
| 护城河 | 生态系统锁定用户 |
| 现金流 | 年自由现金流 1000 亿+ |
| 管理层 | 库克优秀运营能力 |
| 估值 | PE 约 10-12 倍，低估 |
| 回购 | 大规模回购提升 EPS |

#### 投资过程
- **买入**: 2016 年开始建仓
- **加仓**: 持续增持至第一重仓
- **回报**: 成本约 350 亿，现值超 1500 亿

#### 关键启示
1. 能力圈可以缓慢扩展
2. 科技股也可以用价值投资方法
3. 生态系统的护城河极深
4. 承认错误并改变观点

#### 背景
- **发现方式**: 从生活中观察
- **公司阶段**: 快速成长期
- **市场认知**: 尚未充分认可

1. 从生活中发现投资机会
2. 成长股早期识别
3. PEG 估值的有效性

## 经典失败案例

### 案例 1: 巴菲特投资伯克希尔纺织 (1965)

#### 背景
- **买入原因**: 便宜，PB<1
- **行业状况**: 衰退行业
- **后续**: 持续亏损，最终关闭

#### 失败原因
| 问题 | 说明 |
|------|------|
| 行业衰退 | 纺织业整体下滑 |
| 资本密集 | 需要持续投入 |
| 无护城河 | 价格竞争，无定价权 |
| 机会成本 | 资金被占用几十年 |

#### 损失
- **直接损失**: 纺织业务累计亏损
- **机会成本**: 如果投资于其他公司，价值数千亿

#### 关键教训
1. 便宜的烂公司不如好公司
2. 避免衰退行业
3. 机会成本是真实成本
4. 及时止损的重要性

### 案例 2: 巴菲特投资航空股 (2016-2020)

#### 背景
- **买入**: 2016 年开始买入四大航
- **逻辑**: 行业整合，估值低
- **卖出**: 2020 年疫情底部清仓

#### 失败原因
| 问题 | 说明 |
|------|------|
| 行业特性 | 高固定成本，强周期 |
| 黑天鹅 | 疫情导致需求崩溃 |
| 竞争 | 价格战，无定价权 |
| 杠杆 | 高负债，现金流压力大 |

#### 损失
- **买入**: 约 100 亿美元
- **卖出**: 约 40 亿美元
- **亏损**: 约 60 亿美元

#### 关键教训
1. 避免高杠杆周期行业
2. 黑天鹅风险难以预测
3. 承认错误并及时止损
4. 航空业不是好生意

### 案例 3: 孙正义投资 WeWork (2017-2019)

#### 背景
- **投资**: 软银愿景基金重仓
- **金额**: 累计约 100 亿美元
- **估值**: 最高 470 亿美元

#### 失败原因
| 问题 | 说明 |
|------|------|
| 商业模式 | 二房东，无护城河 |
| 治理 | 创始人控制，关联交易 |
| 估值 | 严重高估 |
| 增长 | 烧钱换增长，不可持续 |

#### 损失
- **投资**: 约 100 亿美元
- **现值**: 约 50 亿美元
- **亏损**: 约 50 亿美元+

#### 关键教训
1. 估值不能脱离基本面
2. 公司治理至关重要
3. 增长不等于价值
4. 避免 FOMO 心态

## 案例学习方法

### 1. 深度研究
- 阅读年报和公告
- 分析财务数据
- 了解行业背景
- 研究竞争格局

### 2. 角色扮演
- 假设自己是当时投资者
- 会做出什么决策
- 为什么

### 3. 对比分析
- 成功案例 vs 失败案例
- 不同投资者的选择
- 共同点和差异

### 4. 提炼原则
- 从案例中提取通用原则
- 形成自己的检查清单
- 应用到未来决策
