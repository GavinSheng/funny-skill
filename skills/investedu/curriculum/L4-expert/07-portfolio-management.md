# 投资组合管理实战

**更新时间**: 2026-03-05

## 组合构建流程

### 第一步：确定投资目标
| 要素 | 问题 | 示例 |
|------|------|------|
| 收益目标 | 期望年化回报？ | 8-10% |
| 风险承受 | 最大可接受回撤？ | -20% |
| 投资期限 | 资金使用时间？ | 5 年以上 |
| 流动性需求 | 是否需要随时取用？ | 部分需要 |
| 收入需求 | 是否需要分红收入？ | 年 3% |

### 第二步：资产配置
| 资产类别 | 作用 | 配置比例 |
|----------|------|----------|
| 股票 | 增长引擎 | 40-60% |
| 债券 | 稳定器 | 20-40% |
| 现金 | 流动性和安全垫 | 5-15% |
| 另类 | 分散风险 | 0-20% |

### 第三步：选择投资工具
| 方式 | 优点 | 缺点 | 适合人群 |
|------|------|------|----------|
| 个股 | 回报潜力高 | 风险集中 | 专业投资者 |
| 基金 | 分散风险 | 费用较高 | 大多数投资者 |
| ETF | 费用低、透明 | 被动跟踪 | 指数投资者 |
| 投顾 | 专业服务 | 费用最高 | 高净值客户 |

### 第四步：执行与再平衡
- 分批建仓，避免一次性
- 定期再平衡（季度/年度）
- 记录交易和决策

## 资产配置模型

### 经典 60/40 组合
| 资产 | 比例 | 作用 |
|------|------|------|
| 股票 | 60% | 增长 |
| 债券 | 40% | 稳定 |

**特点**: 简单、经典、适合大多数人

### 耶鲁模式
| 资产 | 比例 |
|------|------|
| 美国股票 | 15% |
| 国际股票 | 25% |
| 债券 | 15% |
| 另类投资 | 45% (PE、VC、对冲基金、房地产) |

**特点**: 高比例另类投资，适合长期资金

### 全天候策略 (达里奥)
| 资产 | 比例 |
|------|------|
| 股票 | 30% |
| 长期国债 | 40% |
| 中期国债 | 15% |
| 黄金 | 7.5% |
| 大宗商品 | 7.5% |

**特点**: 风险平价，适应各种经济环境

### 永久组合 (哈利·布朗)
| 资产 | 比例 | 应对环境 |
|------|------|----------|
| 股票 | 25% | 经济增长 |
| 长期国债 | 25% | 通缩 |
| 现金 | 25% | 经济衰退 |
| 黄金 | 25% | 通胀 |

**特点**: 极简、稳健、低维护

## 再平衡策略

### 再平衡方法
| 方法 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| 定期再平衡 | 每季度/年调整一次 | 简单规律 | 可能错过趋势 |
| 阈值再平衡 | 偏离目标 5% 时调整 | 更精准 | 需要监控 |
| 混合再平衡 | 定期 + 阈值结合 | 平衡两者 | 稍复杂 |

### 再平衡示例
```
初始配置：股票 60%，债券 40%

一年后：股票涨到 70%，债券降到 30%

再平衡操作：
- 卖出 10% 股票
- 买入 10% 债券
- 恢复 60/40 配置
```

### 再平衡原则
- 避免频繁交易（增加成本）
- 考虑税收影响
- 使用新资金调整而非卖出

## 风险管理

### 风险类型
| 风险 | 说明 | 应对 |
|------|------|------|
| 市场风险 | 整体市场下跌 | 资产配置、对冲 |
| 个股风险 | 单只股票暴跌 | 分散投资 |
| 利率风险 | 利率上升债券下跌 | 缩短久期 |
| 通胀风险 | 购买力下降 | 配置实物资产 |
| 流动性风险 | 无法及时变现 | 保持现金 |
| 汇率风险 | 汇率波动 | 对冲或自然对冲 |

### 风险指标
| 指标 | 说明 | 警戒线 |
|------|------|--------|
| 最大回撤 | 历史最大跌幅 | <-30% |
| 波动率 | 年化标准差 | <20% |
| VaR | 95% 置信度最大损失 | <-5%/月 |
| 夏普比率 | 风险调整后收益 | >1 |
| 集中度 | 最大持仓占比 | <20% |

### 风险对冲工具
| 工具 | 用途 | 成本 |
|------|------|------|
| 看跌期权 | 保护下跌 | 期权费 |
| 反向 ETF | 做空市场 | 管理费 |
| 黄金 | 避险 | 机会成本 |
| 现金 | 安全垫 | 通胀损失 |
| VIX 产品 | 波动率对冲 | 高成本 |

## 业绩评估

### 回报指标
| 指标 | 公式 | 说明 |
|------|------|------|
| 绝对回报 | (期末 - 期初)/期初 | 简单收益率 |
| 年化回报 | (1+ 总回报)^(1/n)-1 | 考虑时间 |
| 超额回报 | 组合回报 - 基准回报 | 跑赢市场 |

### 风险调整指标
| 指标 | 公式 | 含义 |
|------|------|------|
| 夏普比率 | (Rp-Rf)/σp | 单位风险的超额回报 |
| 索提诺比率 | (Rp-Rf)/下行偏差 | 只考虑下行风险 |
| 信息比率 | 超额回报/跟踪误差 | 主动管理能力 |
| 卡玛比率 | 年化回报/最大回撤 | 回撤调整后收益 |

### 业绩归因
```
总回报 = 资产配置贡献 + 个股选择贡献 + 交互作用

示例：
- 基准回报：10%
- 组合回报：12%
- 超额回报：+2%
  - 资产配置：+0.5% (超配股票)
  - 个股选择：+1.5% (选股优秀)
  - 交互作用：0%
```

## 投资者行为管理

### 常见错误
| 错误 | 表现 | 对策 |
|------|------|------|
| 追涨杀跌 | 高点买入低点卖出 | 制定计划、定投 |
| 过度交易 | 频繁买卖 | 限制交易次数 |
| 处置效应 | 卖盈持亏 | 定期审视持仓 |
| 确认偏误 | 只听好话 | 寻找反面观点 |
| 过度自信 | 高估能力 | 记录决策日志 |

### 行为清单
- [ ] 我是否有书面投资计划？
- [ ] 我是否按计划执行？
- [ ] 我是否因情绪改变决策？
- [ ] 我是否记录并复盘决策？
- [ ] 我是否定期学习改进？

## 客户报告与沟通

### 报告内容
| 部分 | 内容 | 频率 |
|------|------|------|
| 业绩摘要 | 回报、基准对比 | 月度/季度 |
| 持仓分析 | 前十大持仓、行业分布 | 季度 |
| 市场分析 | 市场回顾、展望 | 季度 |
| 操作说明 | 本期调整及原因 | 发生时 |
| 风险提示 | 风险指标、警示 | 季度 |

### 沟通原则
- 诚实透明，不隐瞒亏损
- 用客户理解的语言
- 管理期望，不承诺收益
- 及时沟通重大变化

## 2026 年组合管理趋势

### 新趋势
- **ESG 整合**: 环境、社会、治理因素纳入决策
- **智能投顾**: AI 驱动的自动化配置
- **个性化**: 基于个人目标的定制方案
