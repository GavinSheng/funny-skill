# 期权策略完整指南

**更新时间**: 2026-03-05
**难度**: L3 专业 → L4 专家
**先修**: 期权基础、希腊字母

## 一、期权策略分类

### 按市场观点分类

| 市场观点 | 策略类型 | 风险特征 |
|----------|----------|----------|
| 方向性 | 看涨/看跌策略 | 单向风险 |
| 波动率 | 做多/做空波动率 | 波动率风险 |
| 中性 | 区间/套利策略 | 有限风险 |
| 对冲 | 保护性策略 | 降低风险 |

### 按复杂度分类

| 级别 | 策略 | 腿数 |
|------|------|------|
| 基础 | 买入/卖出看涨看跌 | 1 腿 |
| 中级 | 价差策略 | 2 腿 |
| 高级 | 组合策略 | 3-4 腿 |
| 专家 | 结构化策略 | 4 腿+ |

## 二、基础策略 (单腿)

### 2.1 买入看涨期权 (Long Call)

**构成**: 买入 1 张看涨期权

**适用场景**: 强烈看涨，预期大涨

**损益分析**:
```
最大收益：无限
最大损失：权利金 (有限)
盈亏平衡点：行权价 + 权利金
```

**希腊字母**:
- Delta: 正 (约 0.3-0.7)
- Gamma: 正
- Theta: 负 (时间衰减不利)
- Vega: 正 (波动率上升有利)

**示例**:
```
标的价格：100 元
买入 Call: 行权价 105, 权利金 3 元

到期情景:
- 标的=90: 损失 3 元 (权利金)
- 标的=105: 损失 3 元 (权利金)
- 标的=108: 盈亏平衡
- 标的=115: 盈利 7 元
- 标的=120: 盈利 12 元
```

---

### 2.2 买入看跌期权 (Long Put)

**构成**: 买入 1 张看跌期权

**适用场景**: 强烈看跌，预期大跌

**损益分析**:
```
最大收益：行权价 - 权利金 (接近无限)
最大损失：权利金 (有限)
盈亏平衡点：行权价 - 权利金
```

**希腊字母**:
- Delta: 负 (约 -0.3 到 -0.7)
- Gamma: 正
- Theta: 负
- Vega: 正

---

### 2.3 卖出看涨期权 (Short Call)

**构成**: 卖出 1 张看涨期权

**适用场景**: 不看涨 (中性或看跌)

**损益分析**:
```
最大收益：权利金 (有限)
最大损失：无限 (标的上涨无上限)
盈亏平衡点：行权价 + 权利金
```

**⚠️ 风险警告**: 裸卖空看涨期权风险极高，需缴纳保证金

---

### 2.4 卖出看跌期权 (Short Put)

**构成**: 卖出 1 张看跌期权

**适用场景**: 不看跌 (中性或看涨)

**损益分析**:
```
最大收益：权利金 (有限)
最大损失：行权价 - 权利金 (大但有限)
盈亏平衡点：行权价 - 权利金
```

**应用**: 想以更低价格买入股票

---

## 三、价差策略 (双腿)

### 3.1 牛市价差 (Bull Call Spread)

**构成**:
- 买入低行权价 Call (K1)
- 卖出高行权价 Call (K2)
- 同期限

**适用场景**: 温和看涨

**损益分析**:
```
最大收益：K2 - K1 - 净权利金
最大损失：净权利金 (有限)
盈亏平衡点：K1 + 净权利金
```

**示例**:
```
标的价格：100 元

买入 Call: K=100, 权利金=5
卖出 Call: K=110, 权利金=2
净权利金：3 元

到期情景:
- 标的=95: 损失 3 元
- 标的=103: 盈亏平衡
- 标的=108: 盈利 5 元
- 标的=115: 最大盈利 7 元
```

**优势**: 降低成本，限制风险
**劣势**: 收益有上限

---

### 3.2 熊市价差 (Bear Put Spread)

**构成**:
- 买入高行权价 Put (K1)
- 卖出低行权价 Put (K2)
- 同期限

**适用场景**: 温和看跌

**损益分析**:
```
最大收益：K1 - K2 - 净权利金
最大损失：净权利金 (有限)
盈亏平衡点：K1 - 净权利金
```

---

### 3.3 日历价差 (Calendar Spread)

**构成**:
- 卖出近期 Call
- 买入远期 Call
- 同 一行权价

**适用场景**: 中性，预期近期波动小

**损益分析**:
```
最大收益：近期权利金 - 远期时间价值衰减
最大损失：净权利金
最佳情景：近期到期时标的=行权价
```

**获利原理**: 近期期权时间价值衰减更快

---

### 3.4 垂直价差 vs 日历价差

| 特征 | 垂直价差 | 日历价差 |
|------|----------|----------|
| 行权价 | 不同 | 相同 |
| 期限 | 相同 | 不同 |
| 主要风险 | 方向风险 | 时间风险 |
| 主要获利 | 方向判断 | 时间衰减差 |

---

## 四、波动率策略

### 4.1 跨式策略 (Long Straddle)

**构成**:
- 买入 Call (行权价 K)
- 买入 Put (行权价 K)
- 同期限

**适用场景**: 预期大波动，但方向不确定

**损益分析**:
```
最大收益：无限 (任一方向大波动)
最大损失：总权利金 (标的=行权价时)
盈亏平衡点：K ± 总权利金
```

**示例**:
```
标的价格：100 元

买入 Call: K=100, 权利金=4
买入 Put: K=100, 权利金=3
总权利金：7 元

到期情景:
- 标的=90: 盈利 3 元 (Put 盈利 10-7)
- 标的=93: 盈亏平衡
- 标的=100: 损失 7 元 (最大损失)
- 标的=107: 盈亏平衡
- 标的=115: 盈利 8 元 (Call 盈利 15-7)
```

**应用**: 财报发布、重大事件前

---

### 4.2 宽跨式策略 (Long Strangle)

**构成**:
- 买入虚值 Call (K2)
- 买入虚值 Put (K1)
- 同期限，K1 < K2

**适用场景**: 预期大波动，成本更低

**损益分析**:
```
最大收益：无限
最大损失：总权利金
盈亏平衡点：K1 - 权利金 或 K2 + 权利金
```

**vs 跨式**: 成本更低，但需要更大波动

---

### 4.3 卖出跨式 (Short Straddle)

**构成**:
- 卖出 Call (行权价 K)
- 卖出 Put (行权价 K)
- 同期限

**适用场景**: 预期无波动，区间震荡

**损益分析**:
```
最大收益：总权利金 (标的=行权价时)
最大损失：无限 (任一方向大波动)
盈亏平衡点：K ± 总权利金
```

**⚠️ 风险警告**: 风险极高，需严格风控

---

### 4.4 卖出宽跨式 (Short Strangle)

**构成**:
- 卖出虚值 Call
- 卖出虚值 Put
- 同期限

**适用场景**: 预期区间震荡，风险略低于 Short Straddle

---

## 五、对冲策略

### 5.1 保护性看跌 (Protective Put)

**构成**:
- 持有股票
- 买入 Put

**适用场景**: 持有股票，担心下跌

**损益分析**:
```
最大收益：无限 (股票上涨)
最大损失：股票买入价 - Put 行权价 + 权利金
盈亏平衡点：股票买入价 + 权利金
```

**效果**: 相当于为股票买保险

**示例**:
```
持有股票：成本 100 元
买入 Put: K=95, 权利金=3

到期情景:
- 标的=80: 损失 8 元 (股票亏 20, Put 赚 15, 权利金 3)
- 标的=95: 损失 8 元 (最小损失)
- 标的=105: 盈利 2 元
- 标的=120: 盈利 17 元
```

---

### 5.2 备兑开仓 (Covered Call)

**构成**:
- 持有股票
- 卖出 Call

**适用场景**: 持有股票，预期温和上涨或震荡

**损益分析**:
```
最大收益：权利金 + (行权价 - 股票成本)
最大损失：股票成本 - 权利金 (股票跌到 0)
盈亏平衡点：股票成本 - 权利金
```

**效果**: 增强收益，但限制上涨空间

**示例**:
```
持有股票：成本 100 元
卖出 Call: K=105, 权利金=2

到期情景:
- 标的=90: 损失 8 元 (股票亏 10, 权利金 2)
- 标的=100: 盈利 2 元 (权利金)
- 标的=105: 盈利 7 元 (股票赚 5+ 权利金 2)
- 标的=110: 盈利 7 元 (股票被行权，收益封顶)
```

**应用**: 长期持股增强收益

---

### 5.3 衣领策略 (Collar)

**构成**:
- 持有股票
- 买入 Put (保护下跌)
- 卖出 Call (支付 Put 费用)

**适用场景**: 保护持仓，降低成本

**损益分析**:
```
最大收益：Call 行权价 - 股票成本
最大损失：股票成本 - Put 行权价
```

**效果**: 锁定收益区间，近乎零成本

---

## 六、高级组合策略

### 6.1 铁鹰式 (Iron Condor)

**构成**:
- 卖出虚值 Put (K1)
- 买入更虚值 Put (K2, K2<K1)
- 卖出虚值 Call (K3)
- 买入更虚值 Call (K4, K4>K3)
- 同期限

**适用场景**: 预期区间震荡

**损益分析**:
```
最大收益：净权利金 (标的在 K1-K3 之间)
最大损失：价差宽度 - 净权利金
盈亏平衡点：K1 - 净权利金 或 K3 + 净权利金
```

**示例**:
```
标的价格：100 元

卖出 Put: K=95, 权利金=2
买入 Put: K=90, 权利金=1
卖出 Call: K=105, 权利金=2
买入 Call: K=110, 权利金=1
净权利金：2 元

到期情景:
- 标的=85: 损失 3 元 (最大损失)
- 标的=93: 盈亏平衡
- 标的=100: 盈利 2 元 (最大盈利)
- 标的=107: 盈亏平衡
- 标的=115: 损失 3 元 (最大损失)
```

**优势**: 高胜率，有限风险
**劣势**: 大波动时亏损

---

### 6.2 铁蝶式 (Iron Butterfly)

**构成**:
- 卖出平值 Put
- 买入虚值 Put
- 卖出平值 Call
- 买入虚值 Call
- 同期限，Call 和 Put 行权价相同

**适用场景**: 强烈预期区间震荡

**vs 铁鹰式**: 盈利区间更窄，但收益更高

---

### 6.3 比率价差 (Ratio Spread)

**构成**:
- 买入 1 张期权
- 卖出 N 张期权 (N>1)
- 不同行权价

**适用场景**: 特定方向温和变动

**风险**: 一侧风险无限，需谨慎

## 七、策略选择矩阵

### 按市场观点选择

| 市场观点 | 波动率 | 推荐策略 | 风险 |
|----------|--------|----------|------|
| 大涨 | 高 | Long Call | 有限 |
| 大涨 | 低 | Long Call, Bull Spread | 有限 |
| 温和涨 | 任意 | Bull Call Spread, Short Put | 有限 |
| 震荡 | 高 | Short Straddle, Iron Condor | 高/有限 |
| 震荡 | 低 | Calendar Spread | 有限 |
| 温和跌 | 任意 | Bear Put Spread | 有限 |
| 大跌 | 高 | Long Put | 有限 |
| 大跌 | 低 | Long Put, Bear Spread | 有限 |

### 按风险偏好选择

| 风险偏好 | 推荐策略 |
|----------|----------|
| 保守 | Covered Call, Protective Put, Collar |
| 稳健 | Bull/Bear Spread, Iron Condor |
| 激进 | Long Call/Put, Straddle |
| 专业 | Short Straddle, Ratio Spread |

## 九、实战应用

### 9.1 财报季策略

**情景**: 公司即将发布财报，预期波动大

**策略选择**:
| 观点 | 策略 |
|------|------|
| 大波动 | Long Straddle/Strangle |
| 超预期上涨 | Long Call |
| 超预期下跌 | Long Put |
| 无波动 | Short Straddle (高风险) |

### 9.2 持股增强策略

**情景**: 长期持有股票，想增强收益

**策略**:
- **温和看涨**: Covered Call
- **担心下跌**: Protective Put
- **锁定区间**: Collar

### 9.3 事件驱动策略

**情景**: 重大事件 (选举、美联储决议)

**策略选择**:
| 预期 | 策略 |
|------|------|
| 大波动 | Long Straddle |
| 区间震荡 | Iron Condor |
| 方向明确 | Long Call/Put |

## 十、风险管理

### 10.1 仓位管理

| 策略类型 | 建议仓位 |
|----------|----------|
| 买方策略 | <5% 资金/策略 |
| 价差策略 | <10% 资金/策略 |
| 卖方策略 | <5% 资金/策略，需保证金 |
| 复杂策略 | <10% 资金/策略 |

### 10.2 止损原则

| 策略 | 止损点 |
|------|--------|
| Long Call/Put | 损失 50% 权利金 |
| 价差策略 | 达到最大损失 50% |
| 卖方策略 | 保证金不足时追加或平仓 |

### 10.3 希腊字母管理

| 风险 | 管理方法 |
|------|----------|
| Delta 风险 | 对冲标的，保持 Delta 中性 |
| Gamma 风险 | 避免临近到期的大头寸 |
| Theta 风险 | 买方控制时间，卖方利用时间 |
| Vega 风险 | 波动率高时卖，低时买 |

## 十一、策略选择清单

开仓前检查:

- [ ] 我的市场观点是什么？(方向 + 波动率)
- [ ] 这个策略是否符合我的观点？
- [ ] 最大损失是多少？我能承受吗？
- [ ] 盈亏平衡点在哪里？
- [ ] 到期日选择是否合适？
- [ ] 流动性是否足够？
- [ ] 保证金是否充足？
- [ ] 有退出计划吗？
