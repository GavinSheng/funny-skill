## 一、衍生品基础

### 1.1 什么是衍生品

**定义**: 衍生品的价值来源于基础资产（标的资产）的价格变动

**基础资产类型**:
- 股票/股指
- 债券/利率
- 外汇
- 大宗商品
- 加密货币

### 1.2 衍生品分类

| 类型 | 交易场所 | 标准化 | 对手方风险 |
|------|----------|--------|------------|
| 期货 | 交易所 | 是 | 低 (清算所担保) |
| 期权 | 交易所 | 是 | 低 |
| 远期 | OTC | 否 | 高 |
| 互换 | OTC | 部分 | 中 - 高 |

### 1.3 衍生品功能

- **套期保值**: 对冲价格风险
- **价格发现**: 反映市场对未来价格的预期
- **投机**: 杠杆交易获取收益
- **套利**: 利用价格差异获利

## 二、期货 (Futures)

### 2.1 期货基础

**定义**: 标准化合约，约定在未来某日以特定价格买卖标的资产

**核心要素**:
- 标的资产
- 合约规模
- 交割日期
- 报价单位
- 最小变动价位

### 2.2 期货定价

**持有成本模型**:
```
期货价格 = 现货价格 + 持有成本 - 持有收益
F = S × e^(r×t)
```

其中:
- F: 期货价格
- S: 现货价格
- r: 无风险利率
- t: 到期时间

### 2.3 基差 (Basis)

```
基差 = 现货价格 - 期货价格
```

**基差风险**: 基差变动导致套保效果不确定

### 2.4 期货交易机制

| 机制 | 说明 |
|------|------|
| 保证金制度 | 初始保证金 + 维持保证金 |
| 每日无负债结算 | 每日盯市 (Mark-to-Market) |
| 涨跌停板 | 限制单日价格波动 |
| 持仓限额 | 限制投机头寸 |
| 强行平仓 | 保证金不足时强制平仓 |

### 2.5 期货类型

| 类型 | 标的 | 示例 |
|------|------|------|
| 商品期货 | 农产品、金属、能源 | 大豆、黄金、原油 |
| 金融期货 | 股指、债券、外汇 | 沪深 300、国债、欧元 |

### 2.6 期货策略

| 策略 | 说明 | 风险 |
|------|------|------|
| 多头套保 | 担心价格上涨，买入期货 | 基差风险 |
| 空头套保 | 担心价格下跌，卖出期货 | 基差风险 |
| 跨期套利 | 不同月份合约价差 | 价差反向 |
| 跨品种套利 | 相关品种价差 | 相关性变化 |
| 投机交易 | 单向押注价格方向 | 高杠杆风险 |

## 三、期权 (Options)

### 3.1 期权基础

**定义**: 赋予持有者在特定时间以特定价格买卖标的资产的权利（非义务）

**基本类型**:
- **看涨期权 (Call)**: 买入权利
- **看跌期权 (Put)**: 卖出权利

**期权分类**:
- **美式期权**: 到期日前任何时间可行权
- **欧式期权**: 仅到期日可行权

### 3.2 期权四要素

| 要素 | 说明 |
|------|------|
| 标的资产 | 期权对应的资产 |
| 行权价 (Strike) | 约定的买卖价格 |
| 到期日 | 期权有效期截止日 |
| 权利金 (Premium) | 期权价格 |

### 3.3 期权价格构成

```
期权价格 = 内在价值 + 时间价值
```

**内在价值**:
- 看涨：max(0, 标的价格 - 行权价)
- 看跌：max(0, 行权价 - 标的价格)

**时间价值**: 随到期日临近而衰减

### 3.4 期权状态

| 状态 | 看涨期权 | 看跌期权 |
|------|----------|----------|
| 实值 | 标的>行权价 | 标的<行权价 |
| 平值 | 标的=行权价 | 标的=行权价 |
| 虚值 | 标的<行权价 | 标的>行权价 |

### 3.5 Black-Scholes 期权定价模型

**公式**:
```
C = S×N(d1) - K×e^(-rt)×N(d2)
P = K×e^(-rt)×N(-d2) - S×N(-d1)
```

其中:
- C: 看涨期权价格
- P: 看跌期权价格
- S: 标的价格
- K: 行权价
- r: 无风险利率
- t: 到期时间
- N(): 标准正态分布函数

### 3.6 期权希腊字母

| 希腊字母 | 含义 | 影响 |
|----------|------|------|
| **Delta (Δ)** | 标的价格变动对期权价格的影响 | 方向风险 |
| **Gamma (Γ)** | 标的价格变动对 Delta 的影响 | Delta 变化速度 |
| **Theta (Θ)** | 时间流逝对期权价格的影响 | 时间衰减 |
| **Vega (ν)** | 波动率变动对期权价格的影响 | 波动率风险 |
| **Rho (ρ)** | 利率变动对期权价格的影响 | 利率风险 |

### 3.7 期权策略

#### 基础策略

| 策略 | 构成 | 预期 | 风险 |
|------|------|------|------|
| 买入看涨 | Long Call | 大涨 | 有限 (权利金) |
| 买入看跌 | Long Put | 大跌 | 有限 (权利金) |
| 卖出看涨 | Short Call | 不涨 | 无限 |
| 卖出看跌 | Short Put | 不跌 | 大 (行权价) |

#### 组合策略

| 策略 | 构成 | 预期 | 风险收益 |
|------|------|------|----------|
| **跨式 (Straddle)** | Long Call + Long Put (同 K) | 大波动 | 有限/无限 |
| **宽跨式 (Strangle)** | Long Call + Long Put (不同 K) | 大波动 | 有限/无限 |
| **牛市价差** | Long Call(K1) + Short Call(K2) | 温和上涨 | 有限/有限 |
| **熊市价差** | Long Put(K1) + Short Put(K2) | 温和下跌 | 有限/有限 |
| **铁鹰式** | 牛市价差 + 熊市价差 | 区间震荡 | 有限/有限 |
| **备兑开仓** | Long Stock + Short Call | 温和上涨 | 大/有限 |
| **保护性看跌** | Long Stock + Long Put | 防下跌 | 有限/无限 |

## 四、远期 (Forwards)

### 4.1 远期基础

**定义**: 非标准化合约，约定在未来某日以特定价格买卖标的资产

**特点**:
- OTC 交易
- 非标准化
- 信用风险高
- 灵活性高

### 4.2 远期 vs 期货

| 特征 | 远期 | 期货 |
|------|------|------|
| 交易场所 | OTC | 交易所 |
| 标准化 | 否 | 是 |
| 清算 | 双边 | 清算所 |
| 保证金 | 协商 | 强制 |
| 流动性 | 低 | 高 |

### 4.3 远期类型

- **外汇远期**: 锁定未来汇率
- **利率远期 (FRA)**: 锁定未来利率
- **商品远期**: 锁定商品价格

## 五、互换 (Swaps)

### 5.1 互换基础

**定义**: 双方约定在未来一系列时间点交换现金流的合约

### 5.2 利率互换 (Interest Rate Swap)

**结构**:
- 一方支付固定利率
- 一方支付浮动利率 (如 LIBOR/SOFR)

**用途**:
- 管理利率风险
- 降低融资成本
- 投机利率变动

### 5.3 货币互换 (Currency Swap)

**结构**:
- 期初交换本金 (不同货币)
- 期间交换利息
- 期末换回本金

**用途**:
- 对冲汇率风险
- 降低外币融资成本

### 5.4 信用违约互换 (CDS)

**定义**: 信用保险合约，买方支付保费，卖方在信用事件发生时赔付

**信用事件**:
- 破产
- 债务违约
- 债务重组

### 5.5 其他互换

| 类型 | 说明 |
|------|------|
| 商品互换 | 交换商品价格相关现金流 |
| 波动率互换 | 交换波动率相关现金流 |
| 总收益互换 | 交换资产总收益 |

## 六、奇异衍生品

### 6.1 路径依赖期权

| 类型 | 特点 |
|------|------|
| 亚式期权 | 收益取决于平均价格 |
| 障碍期权 | 触及障碍价生效/失效 |
| 回望期权 | 收益取决于期间最高/最低价 |

### 6.2 多资产期权

| 类型 | 特点 |
|------|------|
| 篮子期权 | 标的为一篮子资产 |
| 彩虹期权 | 收益取决于多个资产表现 |
| 交换期权 | 交换两种资产的权利 |

### 6.3 结构化产品

| 产品 | 构成 |
|------|------|
| 可转债 | 债券 + 转股期权 |
| 雪球 | 期权组合 + 自动赎回 |
| 保本票据 | 债券 + 期权 |

## 七、衍生品风险管理

### 7.1 风险类型

| 风险 | 说明 | 管理方法 |
|------|------|----------|
| 市场风险 | 价格变动损失 | 对冲、限额 |
| 信用风险 | 对手方违约 | 抵押品、净额结算 |
| 流动性风险 | 无法平仓 | 分散、限额 |
| 操作风险 | 操作失误 | 内控、系统 |
| 模型风险 | 定价模型错误 | 验证、备份 |

### 7.2 风险指标

| 指标 | 说明 |
|------|------|
| VaR (风险价值) | 一定置信度下最大损失 |
| 压力测试 | 极端情景下损失 |
| 敏感性分析 | 参数变动影响 |
|  Greeks | 期权风险敞口 |

### 7.3 保证金管理

- **初始保证金**: 开仓时缴纳
- **维持保证金**: 最低保证金水平
- **追加保证金**: 低于维持时追加
- **变动保证金**: 每日盈亏结算

## 八、衍生品监管

### 8.1 监管框架

| 地区 | 监管机构 | 主要法规 |
