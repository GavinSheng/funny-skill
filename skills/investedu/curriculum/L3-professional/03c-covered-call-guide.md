## 一、Covered Call 策略回顾

### 策略构成
```
Covered Call = 持有股票 + 卖出看涨期权
```

### 适用场景
- ✅ 长期持有股票
- ✅ 预期温和上涨或震荡
- ✅ 愿意在行权价卖出股票
- ❌ 预期大涨 (会限制收益)
- ❌ 预期大跌 (保护有限)

### 损益特征
```
最大收益 = 权利金 + (行权价 - 股票成本)
最大损失 = 股票成本 - 权利金 (股票跌到 0)
盈亏平衡点 = 股票成本 - 权利金
```

## 二、选股核心标准

### 2.1 基本面标准 ⭐⭐⭐⭐⭐

| 标准 | 要求 | 原因 |
|------|------|------|
| **优质公司** | 行业龙头、护城河深 | 长期持有信心 |
| **稳定盈利** | 连续 3-5 年盈利 | 降低暴跌风险 |
| **合理估值** | PE 低于历史平均 | 安全边际 |
| **分红记录** | 有稳定分红 | 额外收入来源 |
| **流动性好** | 日均成交>100 万股 | 期权流动性好 |

### 2.2 技术面标准 ⭐⭐⭐⭐

| 标准 | 要求 | 原因 |
|------|------|------|
| **趋势** | 上升通道或横盘 | 避免下跌趋势 |
| **支撑位** | 接近强支撑 | 下跌空间有限 |
| **阻力位** | 上方有明确阻力 | 选择行权价参考 |
| **波动率** | 中等 (30%-60%) | 权利金适中 |
| **RSI** | 40-60 区间 | 不过热也不过冷 |

### 2.3 期权流动性标准 ⭐⭐⭐⭐⭐

| 标准 | 要求 | 原因 |
|------|------|------|
| **买卖价差** | <0.5 元 | 交易成本低 |
| **未平仓合约** | >1000 张 | 流动性好 |
| **成交量** | 日均>100 张 | 容易成交 |
| **行权价间隔** | 密集 (2.5-5 元) | 选择灵活 |
| **到期日选择** | 丰富 (周/月/季) | 策略灵活 |

## 三、选股量化筛选器

### 3.1 基础筛选条件

```
市值 > 100 亿
日均成交量 > 100 万股
PE < 行业平均
连续 3 年盈利
有期权交易
```

### 3.2 进阶筛选条件

```
隐含波动率 (IV) > 30%
Beta < 1.5 (避免过度波动)
股息率 > 1% (加分项)
机构持股比例 > 30%
分析师覆盖 > 5 人
```

### 3.3 期权筛选条件

```
Call 期权买卖价差 < 0.5 元
近月未平仓合约 > 1000 张
行权价覆盖 +/-20% 区间
有周期权 (更佳)
```

## 五、行权价选择技巧

### 5.1 行权价选择矩阵

| 市场预期 | 推荐行权价 | 权利金 | 被行权概率 |
|----------|------------|--------|------------|
| 强烈看涨 | 远离现价 (OTM 20%+) | 低 | 低 |
| 温和看涨 | 适度 OTM (5-10%) | 中 | 中 |
| 中性震荡 | 轻度 OTM (2-5%) | 中高 | 中高 |
| 温和看跌 | 平值或轻度 ITM | 高 | 高 |

### 5.2 Delta 选择法

| Delta | 含义 | 被行权概率 | 适用场景 |
|-------|------|------------|----------|
| 0.20-0.30 | 深度虚值 | ~25% | 强烈看涨，不想被行权 |
| 0.30-0.40 | 虚值 | ~35% | 温和看涨 (推荐) |
| 0.40-0.50 | 平值附近 | ~50% | 中性，最大化权利金 |
| 0.50-0.60 | 实值 | ~60% | 愿意卖出股票 |

**推荐**: Delta 0.30-0.40，平衡收益与被行权风险

### 5.3 技术位选择法

```
阻力位法:
- 查看上方第一个阻力位
- 选择阻力位附近的行权价
- 理由：股价难突破阻力，被行权概率低

支撑位法:
- 查看下方第一个支撑位
- 确保行权价高于支撑位
- 理由：即使下跌，也有缓冲
```

### 5.4 示例分析

```
股票：贵州茅台
现价：1700 元

上方阻力位：1800 元
下方支撑位：1600 元

可选行权价:
1. 1750 元 (OTM 3%) - Delta 0.40 - 权利金 40 元
2. 1800 元 (OTM 6%) - Delta 0.30 - 权利金 25 元
3. 1850 元 (OTM 9%) - Delta 0.20 - 权利金 15 元

推荐选择:
- 想被行权：选 1750 元
- 不想被行权：选 1800 元
- 强烈看涨：选 1850 元
```

## 六、到期日选择技巧

### 6.1 到期日选择对比

| 到期日 | 权利金 | 时间衰减 | 灵活性 | 推荐度 |
|--------|--------|----------|--------|--------|
| 周期权 (7 天) | 低 | 最快 | 最高 | ⭐⭐⭐ |
| 月期权 (30 天) | 中 | 快 | 高 | ⭐⭐⭐⭐⭐ |
| 季期权 (90 天) | 高 | 中等 | 中 | ⭐⭐⭐ |
| 长期 (180 天+) | 很高 | 慢 | 低 | ⭐⭐ |

### 6.2 推荐选择

**最佳**: 30-45 天到期
- 时间价值衰减最快阶段
- 权利金适中
- 灵活性高
- 可每月滚动操作

### 6.3 滚动策略

```
月度滚动:
- 每月卖出一次 Call
- 到期后继续卖出下月 Call
- 年化权利金收入：10-20%

周度滚动:
- 每周卖出一次 Call
- 更灵活，但交易成本高
- 适合有时间盯盘的投资者
```

## 七、波动率选择技巧

### 7.1 IV 百分位法

| IV 百分位 | 含义 | 策略 |
|-----------|------|------|
| <20% | 波动率极低 | 不适合 (权利金太低) |
| 20-40% | 波动率低 | 谨慎 (权利金偏低) |
| 40-60% | 波动率适中 | 适合 (推荐) |
| 60-80% | 波动率高 | 很适合 (权利金高) |
| >80% | 波动率极高 | 谨慎 (可能有重大事件) |

### 7.2 IV vs HV 比较

```
隐含波动率 (IV) > 历史波动率 (HV):
- 期权价格偏贵
- 适合卖出期权 (Covered Call)

隐含波动率 (IV) < 历史波动率 (HV):
- 期权价格便宜
- 不适合卖出期权
```

### 7.3 波动率时机

**最佳时机**:
- 财报后 1-2 周 (IV 回落)
- 市场恐慌后 (IV 高企)
- 横盘整理期 (IV 稳定)

**避免时机**:
- 财报前 (IV 太高，可能被行权)
- 重大事件前 (不确定性高)
- 趋势明确大涨时 (机会成本)

## 八、持仓管理技巧

### 8.1 仓位控制

| 投资者类型 | Covered Call 仓位 | 纯持股仓位 |
|------------|------------------|------------|
| 保守型 | 30-50% | 50-70% |
| 平衡型 | 50-70% | 30-50% |
| 激进型 | 70-100% | 0-30% |

**建议**: 不要 100% 持仓做 Covered Call，保留部分纯持股

### 8.2 分散原则

```
- 单只股票 < 20% 总资金
- 单个行业 < 40% 总资金
- 至少持有 3-5 只股票
- 跨行业分散
```

### 8.3 动态调整

| 情景 | 操作 |
|------|------|
| 股价大涨接近行权价 | 平仓或向上滚动 |
| 股价大跌 | 继续持有或向下滚动 |
| 波动率大幅下降 | 考虑平仓 |
| 基本面恶化 | 平仓止损 |

## 九、实战案例

### 案例 1: 蓝筹股 Covered Call

```
标的：贵州茅台 (600519)
建仓时间：2026-01-15
股票成本：1700 元/股
持仓：100 股

期权选择:
- 行权价：1800 元 (OTM 6%)
- 到期日：2026-02-15 (30 天)
- 权利金：25 元/股

情景分析:
1. 到期股价 1750 元 (<1800)
   - 期权作废，赚取 25 元权利金
   - 股票浮盈 50 元
   - 总收益：75 元 (4.4%)

2. 到期股价 1850 元 (>1800)
   - 被行权，股票以 1800 元卖出
   - 股票盈利：100 元
   - 权利金：25 元
   - 总收益：125 元 (7.4%)
   - 错失额外 50 元涨幅

3. 到期股价 1600 元 (下跌)
   - 期权作废，赚取 25 元权利金
   - 股票亏损：100 元
   - 净亏损：75 元 (-4.4%)
   - 比纯持股少亏 25 元
```

### 案例 2: ETF Covered Call

```
标的：沪深 300ETF (510300)
建仓时间：2026-01-15
ETF 成本：4.00 元/份
持仓：10000 份

期权选择:
- 行权价：4.20 元 (OTM 5%)
- 到期日：2026-02-15 (30 天)
- 权利金：0.08 元/份

月度收益:
- 权利金收入：800 元
- 年化收益率：0.08/4.00 × 12 = 24%

优势:
- ETF 波动小于个股
- 分散风险
- 期权流动性好
```

## 十、常见错误与避免

### ❌ 错误 1: 选择不合适的股票

**错误**: 选择妖股、题材股做 Covered Call
**后果**: 股票可能暴跌，权利金无法弥补
**避免**: 只选择优质蓝筹、ETF

### ❌ 错误 2: 行权价太近

**错误**: 选择平值或实值 Call
**后果**: 容易被行权，错过大涨
**避免**: 选择 Delta 0.30-0.40 的虚值 Call

### ❌ 错误 3: 到期日太短

**错误**: 选择周期权
**后果**: 管理成本高，容易出错
**避免**: 选择 30-45 天月期权

### ❌ 错误 4: 100% 仓位

**错误**: 全部持仓做 Covered Call
**后果**: 完全错失大涨机会
**避免**: 保留 30-50% 纯持股

### ❌ 错误 5: 财报前卖出

**错误**: 财报前卖出 Call
**后果**: 财报后大幅波动，风险不可控
**避免**: 财报后 1-2 周再操作

### ❌ 错误 6: 不设置止损

**错误**: 股票大跌不止损
**后果**: 权利金无法弥补股票损失
**避免**: 股票跌破支撑位考虑止损
