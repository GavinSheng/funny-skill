# Covered Call 选股技巧实战指南

**更新时间**: 2026-03-05
**难度**: L3 专业阶段
**先修**: Covered Call 基础、期权基础

---

## 一、Covered Call 策略回顾

### 策略构成
```
Covered Call = 持有股票 + 卖出看涨期权
```

### 适用场景
- 长期持有股票
- 预期温和上涨或震荡
- 想增强收益 (权利金收入)
- 愿意在行权价卖出股票

### 收益特征
```
最大收益 = 权利金 + (行权价 - 股票成本)
最大损失 = 股票成本 - 权利金 (股票跌到 0)
盈亏平衡点 = 股票成本 - 权利金
```

## 二、选股核心标准

### 2.1 基础筛选条件

| 条件 | 标准 | 原因 |
|------|------|------|
| **流动性** | 日成交量>100 万股 | 确保期权流动性 |
| **市值** | >100 亿美元 | 降低个股风险 |
| **股价** | 20-500 元 | 适中价格，期权活跃 |
| **期权链** | 有周度/月度期权 | 灵活选择到期日 |
| **行业** | 稳定行业优先 | 降低波动风险 |

### 2.2 波动率要求

| 指标 | 理想范围 | 说明 |
|------|----------|------|
| **历史波动率 (HV)** | 20%-40% | 波动太低权利金少，太高风险大 |
| **隐含波动率 (IV)** | 30%-50% | IV 高时权利金丰厚 |
| **IV Rank** | >30% | IV 处于历史较高分位 |
| **IV vs HV** | IV > HV | 隐含波动率溢价 |

**为什么重要**:
- 波动率决定权利金大小
- IV 高时卖出期权更划算
- 但避免波动率过高的高风险股票

### 2.3 基本面要求

| 指标 | 标准 | 原因 |
|------|------|------|
| **市盈率 (PE)** | 10-25 倍 | 估值合理 |
| **市净率 (PB)** | 1-5 倍 | 避免高估值 |
| **股息率** | 1%-5% | 有分红增强收益 |
| **ROE** | >10% | 盈利能力良好 |
| **负债率** | <60% | 财务稳健 |
| **盈利增长** | 稳定或温和增长 | 避免大起大落 |

### 2.4 技术面要求

| 指标 | 理想状态 | 原因 |
|------|----------|------|
| **趋势** | 上升通道或盘整 | 避免下降趋势 |
| **支撑位** | 接近当前价 | 下行保护 |
| **阻力位** | 上方清晰 | 帮助选择行权价 |
| **RSI** | 40-60 | 不过热也不过冷 |
| **均线** | 股价在 200 日均线上方 | 长期趋势向上 |

## 三、选股量化筛选器

### 3.1 基础筛选公式

```
筛选条件:
1. 市值 > 100 亿美元
2. 股价 20-500 元
3. 日成交量 > 100 万股
4. 有期权交易
5. IV Rank > 30%
6. 股息率 1%-5%
7. PE 10-25 倍
8. ROE > 10%
9. 股价 > 200 日均线
10. RSI 40-60
```

### 3.2 评分系统

给每只股票打分 (满分 100):

| 因素 | 权重 | 评分标准 |
|------|------|----------|
| **期权权利金** | 30 分 | 月权利金/股价 >2% 得 30 分，1-2% 得 20 分 |
| **波动率** | 20 分 | IV Rank 30-50% 得 20 分 |
| **基本面** | 25 分 | PE/PB/ROE 综合评分 |
| **技术面** | 15 分 | 趋势、支撑、RSI 评分 |
| **行业稳定性** | 10 分 | 防御行业高分 |

**建议**: 只选择总分>70 分的股票

## 四、行业选择

### 4.1 适合 Covered Call 的行业

| 行业 | 适合度 | 原因 | 代表股票 |
|------|--------|------|----------|
| **银行** | ⭐⭐⭐⭐⭐ | 波动适中，分红高 | 工商银行、招商银行 |
| **公用事业** | ⭐⭐⭐⭐⭐ | 稳定，波动低 | 长江电力 |
| **消费必需品** | ⭐⭐⭐⭐ | 防御性，稳定 | 贵州茅台、伊利股份 |
| **电信** | ⭐⭐⭐⭐ | 高分红，波动适中 | 中国移动 |
| **能源** | ⭐⭐⭐ | 周期性强，但权利金高 | 中国石油 |
| **房地产 (REITs)** | ⭐⭐⭐⭐ | 高分红，波动适中 | 各类 REITs |

| 行业 | 原因 |
|------|------|
| 生物制药 | 波动太大，二元事件风险 |

| 财务困境 | 破产风险 |
| 并购目标 | 可能被收购导致大幅波动 |

## 五、具体选股步骤

### 步骤 1: 初筛股票池

使用筛选器找出候选股票:

```
候选池要求:
- 市值>100 亿美元
- 有期权交易
- IV Rank > 30%
- 股息率>1%
```

### 步骤 2: 检查期权链

对每只候选股票检查:

```
期权链检查:
□ 是否有月度期权？
□ 买卖价差是否合理 (<5%)?
□ 成交量是否活跃？
□ 不同行权价选择是否丰富？
```

### 步骤 3: 计算预期收益

对每只股票计算:

```
预期年化收益 = (权利金 + 股息) / 股票成本 × 12/月数

示例:
股票成本：100 元
月权利金：2 元 (2%)
年股息：3 元 (3%)
预期年化 = (2×12 + 3) / 100 = 27%
```

### 步骤 4: 风险评估

检查风险因素:

```
风险检查:
□ 近期是否有财报？(避免财报前卖出)
□ 是否有重大事件？(产品发布、诉讼等)
□ 技术面是否破位？
□ 行业是否有利空？
□ 大盘是否处于高位？
```

### 步骤 5: 最终选择

选择 3-5 只股票构建组合:

```
组合原则:
- 分散不同行业 (至少 3 个行业)
- 单只股票<20% 仓位
- 总仓位控制在 60-80%
- 保留现金应对下跌
```

## 六、行权价选择技巧

### 6.1 行权价选择策略

| 策略 | 行权价 | 权利金 | 被行权概率 | 适合场景 |
|------|--------|--------|------------|----------|
| **保守型** | OTM 10-15% | 低 | 低 | 强烈看涨，不想被行权 |
| **平衡型** | OTM 5-10% | 中 | 中 | 温和看涨，可接受部分行权 |
| **激进型** | ATM 或 OTM 0-5% | 高 | 高 | 不看涨，想最大化权利金 |

### 6.2 Delta 选择法

用 Delta 指导行权价选择:

| Delta | 含义 | 被行权概率 | 推荐 |
|-------|------|------------|------|
| 0.10-0.20 | 深度虚值 | 10-20% | 保守策略 |
| 0.20-0.30 | 虚值 | 20-30% | 平衡策略 ⭐ |
| 0.30-0.40 | 轻度虚值 | 30-40% | 激进策略 |
| 0.50 | 平值 | 50% | 不推荐 |

**推荐**: 选择 Delta 0.20-0.30 的行权价

### 6.3 技术位选择法

结合技术面选择行权价:

```
行权价选择:
1. 找出上方阻力位
2. 选择阻力位上方的行权价
3. 确保有足够上涨空间

示例:
当前股价：100 元
阻力位：108 元
推荐行权价：110 元 (OTM 10%)
```

## 七、到期日选择技巧

### 7.1 到期日选择

| 到期日 | 优点 | 缺点 | 推荐度 |
|--------|------|------|--------|
| **周度 (7 天)** | 快速周转，灵活 | 权利金少，频繁操作 | ⭐⭐ |
| **月度 (30 天)** | 权利金适中，操作频率适中 | 平衡 | ⭐⭐⭐⭐⭐ |
| **季度 (60-90 天)** | 权利金高，操作少 | 资金占用久，灵活性差 | ⭐⭐⭐ |

**推荐**: 月度期权 (30-45 天)

### 7.2 时间衰减规律

```
时间价值衰减速度:
- 到期前 60 天：缓慢
- 到期前 30 天：加速
- 到期前 14 天：快速
- 到期前 7 天：极速

最佳卖出时点：到期前 30-45 天
```

## 八、建仓时机

### 8.1 最佳建仓时机

| 时机 | 说明 | 推荐度 |
|------|------|--------|
| **财报后 1-2 周** | 波动率下降，股价稳定 | ⭐⭐⭐⭐⭐ |
| **IV 高位时** | 权利金丰厚 | ⭐⭐⭐⭐⭐ |
| **技术支撑位** | 下行风险小 | ⭐⭐⭐⭐ |
| **市场恐慌后** | IV 高，股价低 | ⭐⭐⭐⭐ |
| **分红除息前** | 注意除息影响 | ⭐⭐⭐ |

### 8.2 避免的时机

| 时机 | 原因 |
|------|------|
| 财报前 1 周 | 波动大，不确定性高 |
| 重大事件前 | 风险不可控 |
| 股价连续大涨后 | 可能回调 |
| 技术破位时 | 下降趋势中 |
| IV 极低时 | 权利金太少 |

## 九、实战案例

### 案例 1: 银行股 Covered Call

**股票**: 招商银行 (600036.SH)

**建仓时数据**:
```
日期：2026-02-01
股价：35 元
持仓：1000 股
成本：35,000 元

期权选择:
行权价：37 元 (OTM 5.7%)
到期日：2026-03-01 (30 天)
权利金：0.8 元/股
总收入：800 元 (2.3%)
```

**情景分析**:
```
情景 1: 到期股价=36 元
- 期权作废，保留权利金 800 元
- 股票盈利 1000 元
- 总盈利：1800 元 (5.1%)

情景 2: 到期股价=38 元
- 期权被行权，股票以 37 元卖出
- 股票盈利 2000 元 + 权利金 800 元
- 总盈利：2800 元 (8%)

情景 3: 到期股价=33 元
- 期权作废，保留权利金 800 元
- 股票亏损 2000 元
- 总亏损：1200 元 (-3.4%)
```

**年化收益估算**: 2.3% × 12 = 27.6%

## 十、风险管理

### 10.1 仓位管理

| 风险等级 | 单只股票 | 单行业 | 总仓位 |
|----------|----------|--------|--------|
| 保守 | <10% | <30% | <50% |
| 平衡 | <15% | <40% | <70% |
| 激进 | <20% | <50% | <90% |

**推荐**: 平衡型配置

### 10.2 下行保护

| 方法 | 说明 | 成本 |
|------|------|------|
| **止损单** | 股价跌破支撑位止损 | 无成本，可能假突破 |
| **买入 Put** | Collar 策略 | 权利金成本 |
| **现金储备** | 保留 20-30% 现金 | 机会成本 |
| **分散投资** | 多只股票分散 | 无成本 |

### 10.3 应对策略

**情景 1: 股价大涨，即将被行权**
```
选择:
1. 接受行权，获利了结 ✓ (推荐)
2. 回购期权，展期到更高行权价
3. 回购期权，展期到更晚到期日
```

**情景 2: 股价大跌**
```
选择:
1. 持有股票，继续卖出 Call ✓ (推荐)
2. 止损卖出股票
3. 买入 Put 保护 (Collar)
```

**情景 3: 股价横盘**
```
选择:
1. 继续卖出 Call，收取权利金 ✓ (最佳情景)
```

## 十一、常见错误

### ❌ 错误 1: 选择波动率过高的股票
```
错误：选择生物制药股，IV>80%
后果：股价大波动，可能大幅亏损
正确：选择 IV 30-50% 的稳定股票
```

### ❌ 错误 2: 行权价选择太近
```
错误：选择 OTM 0-2% 的行权价
后果：频繁被行权，失去股票
正确：选择 OTM 5-10% 的行权价
```

### ❌ 错误 3: 财报前卖出 Call
```
错误：财报前 1 周卖出 Call
后果：财报后股价大波动，风险不可控
正确：财报后 1-2 周再卖出
```

### ❌ 错误 4: 仓位过重
```
错误：单只股票>30% 仓位
后果：个股风险过大
正确：单只股票<15% 仓位
```

### ❌ 错误 5: 忽视下行风险
```
错误：只想着收权利金，不担心股票下跌
后果：股票大跌，权利金无法弥补
正确：选择基本面好的股票，设置止损
```

## 总结

### Covered Call 选股核心要点

```
1. 选股标准：基本面好 + 波动率适中 + 期权活跃
2. 行权价：OTM 5-10%, Delta 0.20-0.30
3. 到期日：月度期权 (30-45 天)
4. 建仓时机：财报后、IV 高位、技术支撑
5. 仓位管理：单只<15%, 分散行业
6. 预期收益：年化 20-30%
```

### 成功公式

```
成功 = 好股票 × 合适行权价 × 正确时机 × 严格风控
```
