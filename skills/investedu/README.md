## 📖 简介

InvestEdu 是一个标准化的投资知识教育技能，提供从入门到专业的完整学习体系。参照 CFA 知识框架构建，涵盖股票、债券、基金、估值、宏观经济、行业分析等全领域投资知识。

## ✨ 特性

- 📚 **完整体系**: L1-L4 四个阶段，55+ 课程模块
- 🎯 **个性规划**: 根据目标自动生成学习计划
- 💡 **知识问答**: 本地 + 联网搜索解答问题
- 📝 **水平测试**: 阶段性测试检验学习效果
- 📊 **进度跟踪**: 记录学习进度和成就
- 🔄 **持续更新**: 季度更新策略，与时俱进

## 🚀 快速开始

### 前置要求

- Bash
- jq (JSON 处理)
- OpenClaw v1.0+

### 安装检查

```bash
# 检查依赖
which bash && which jq

# 如缺少 jq，安装:
# Ubuntu/Debian: sudo apt-get install jq
# macOS: brew install jq
```

### 基本使用

```bash
cd ~/.openclaw/workspace/skills/investedu/scripts

# 1. 查看知识体系
./show-framework.sh

# 2. 评估当前水平
./assess-level.sh

# 3. 规划学习路径
./plan-learning.sh "股票投资"

# 4. 知识问答
./knowledge-qa.sh "什么是 PE 估值"

# 5. 知识测试
./quiz.sh valuation

# 6. 查看进度
./progress.sh
```
