## Q61: PE 估值法怎么用？多少倍算合理？

### 答案

### PE (市盈率) 定义

**公式**:
```
PE = 股价 / 每股收益 (EPS)
   = 总市值 / 净利润
```

**含义**: 投资者愿意为每 1 元净利润支付多少价格
**通俗理解**: 按当前盈利水平，多少年能回本

---

### PE 的计算方式

#### 1. 静态 PE (LYR)
```
PE(静态) = 当前市值 / 去年净利润
缺点：数据滞后
```

#### 2. 滚动 PE (TTM) ⭐推荐
```
PE(TTM) = 当前市值 / 过去 4 季度净利润之和
优点：数据最新，最常用
```

#### 3. 动态 PE
```
PE(动态) = 当前市值 / 预测今年净利润
缺点：预测可能不准
```

---

### PE 多少倍算合理？

**没有绝对标准，要看**:

#### 1. 行业差异
```
| 行业 | 合理 PE 范围 | 说明 |
|------|-------------|------|
| 银行 | 4-8 倍 | 增长慢，周期性强 |
| 保险 | 8-15 倍 | 内含价值增长 |
| 地产 | 5-10 倍 | 周期性强 |
| 家电 | 10-20 倍 | 稳定增长 |
| 白酒 | 20-40 倍 | 高毛利，稳定 |
| 医药 | 25-50 倍 | 高成长 |
| 科技 | 30-60 倍 | 高成长，高波动 |
| 互联网 | 20-50 倍 | 看用户增长 |
| 新能源 | 30-80 倍 | 高成长高估值 |
```

#### 2. 成长性 (PEG 指标)
```
PEG = PE / 净利润增长率

✅ PEG < 1: 低估 (增速快于 PE)
✅ PEG = 1-1.5: 合理
❌ PEG > 2: 高估 (增速跟不上 PE)

例子:
- 公司 A: PE=30 倍，增速 30% → PEG=1，合理
- 公司 B: PE=50 倍，增速 20% → PEG=2.5，高估
- 公司 C: PE=20 倍，增速 30% → PEG=0.67，低估
```

#### 3. 历史估值区间
```
查看公司历史 PE Band:
- 当前 PE 处于历史低位 (20% 分位以下) → 可能低估
- 当前 PE 处于历史中位 (40%-60% 分位) → 合理
- 当前 PE 处于历史高位 (80% 分位以上) → 可能高估
```

#### 4. 利率环境
```
- 利率低：PE 可以更高 (资金成本低)
- 利率高：PE 应该更低 (资金成本高)

简化公式：合理 PE ≈ 1 / 无风险利率
- 利率 3%: 合理 PE ≈ 33 倍
- 利率 5%: 合理 PE ≈ 20 倍
```

---

### PE 估值法的适用场景

#### ✅ 适合用 PE 的情况
```
- 盈利稳定的公司
- 成熟行业龙头
- 周期性行业的正常期
- 消费、医药等稳定增长行业
```

#### ❌ 不适合用 PE 的情况
```
- 亏损公司 (PE 为负)
- 周期性行业的高点/低点
- 初创公司 (还没盈利)
- 利润波动大的公司
- 一次性收益/损失影响大的
```

---

### PE 估值的陷阱

#### 1. 周期股陷阱
```
周期股特点：盈利波动大
- 行业高点：利润高→PE 低→看似便宜实则贵
- 行业低点：利润低→PE 高→看似贵实则便宜

例子：钢铁、煤炭、航运
```

#### 2. 一次性收益陷阱
```
公司卖资产获得大额收益→利润暴增→PE 骤降
→ 看似便宜，实际不可持续
```

#### 3. 会计调节陷阱
```
公司通过会计手段调节利润
→ PE 看起来合理，实际盈利质量差
```

#### 4. 成长陷阱
```
高 PE 买高成长公司
→ 一旦成长不及预期，戴维斯双杀
→ PE 下降 + 股价下跌
```

---

### PE 估值实战步骤

1. **确认公司是否适合 PE 估值**
   - 是否持续盈利？
   - 盈利是否稳定？

2. **计算 PE(TTM)**
   - 使用滚动 PE

3. **横向对比**
   - 与同行比较
   - 与行业平均比较

4. **纵向对比**
   - 与历史 PE 区间比较
   - 当前处于什么分位？

5. **结合成长性**
   - 计算 PEG
   - 增速能否支撑当前 PE？

6. **综合判断**
   - 低估：可以考虑买入
   - 合理：持有或定投
   - 高估：谨慎或卖出

---

## Q62: PB 估值法适合什么行业？

### 答案

### PB (市净率) 定义

**公式**:
```
PB = 股价 / 每股净资产
   = 总市值 / 净资产
```

**含义**: 股价相对净资产的溢价程度
**通俗理解**: 你花多少钱买 1 元净资产

---

### PB 的适用场景

#### ✅ 最适合的行业

##### 1. 银行业 ⭐⭐⭐⭐⭐
```
原因:
- 资产主要是贷款，价值清晰
- 净资产稳定增长
- 盈利波动相对小

合理 PB 范围:
- 国有大行：0.5-0.8 倍
- 股份行：0.6-1.0 倍
- 城商行：0.8-1.5 倍
```

##### 2. 保险业 ⭐⭐⭐⭐
```
原因:
- 资产主要是投资资产
- 内含价值可参考净资产

合理 PB 范围:
- 传统险为主：1-2 倍
- 寿险为主：1.5-3 倍
```

##### 3. 证券业 ⭐⭐⭐
```
原因:
- 资产主要是金融资产
- 周期性较强

合理 PB 范围:
- 正常：1.5-3 倍
- 牛市：3-5 倍
- 熊市：1-1.5 倍
```

##### 4. 重资产行业 ⭐⭐⭐
```
- 钢铁：0.8-1.5 倍
- 煤炭：1-2 倍
- 化工：1-2 倍
- 建材：1-2 倍
```

#### ❌ 不适合的行业

```
- 轻资产行业：互联网、软件 (净资产少)
- 服务业：人力资本为主
- 高 ROE 公司：净资产不能反映真实价值
- 亏损公司：净资产可能持续减少
```

---

### PB 与 ROE 的关系

**核心逻辑**: 高 ROE 应该享受高 PB

**公式**:
```
合理 PB = ROE × 合理 PE
```

**参考**:
```
| ROE | 合理 PB 范围 |
|-----|-----------|
| <10% | 0.8-1.2 倍 |
| 10%-15% | 1.2-2 倍 |
| 15%-20% | 2-3 倍 |
| >20% | 3 倍以上 |
```

**例子**:
```
银行 A: ROE=12%, PB=0.7 倍 → 可能低估
银行 B: ROE=12%, PB=1.5 倍 → 可能高估

茅台：ROE=30%, PB=8 倍 → 合理 (高 ROE 支撑高 PB)
```

---

### PB 估值实战

#### 1. 判断低估/高估
```
PB < 1: 破净，可能低估 (但要分析原因)
PB = 1-2: 合理区间
PB > 3: 高估 (高 ROE 公司除外)
```

#### 2. 破净股分析
```
破净 (PB<1) 可能的原因:
✅ 真低估：市场过度悲观
❌ 价值陷阱：
   - 资产质量差 (坏账多)
   - 持续亏损 (净资产会减少)
   - 行业衰退
```

#### 3. 结合其他指标
```
- PB + ROE: 高 ROE 低 PB 最佳
- PB + PE: 双低更安全
- PB + 股息率：高股息破净股有吸引力
```

---

### PB 估值的局限性

```
❌ 不能反映无形资产价值 (品牌、专利)
❌ 不能反映盈利能力差异
❌ 净资产可能失真 (资产减值不充分)
❌ 不适合轻资产公司
```

---

### PB 估值最佳实践

1. **选对行业**: 银行、保险、重资产
2. **看 ROE**: 高 ROE 配高 PB
3. **看历史**: 当前 PB 处于什么分位
4. **看同行**: 与可比公司比较
5. **看资产质量**: 净资产是否真实
6. **结合 PE**: 双低更安全

---

## Q63: DCF 估值法是什么？怎么用？

### 答案

### DCF (现金流折现) 定义

**核心思想**: 公司的价值等于未来所有现金流的现值之和

**基本原理**: 今天的 100 元 > 明天的 100 元 (时间价值)

---

### DCF 公式

```
公司价值 = Σ [FCFt / (1+r)^t] + 终值 / (1+r)^n

其中:
- FCFt: 第 t 年的自由现金流
- r: 折现率 (要求的回报率)
- t: 年份 (1, 2, 3...n)
- 终值：n 年后的持续价值
```

---

### DCF 估值步骤

#### 第 1 步：预测自由现金流 ⭐⭐⭐⭐⭐

**自由现金流 (FCF)**:
```
FCF = 经营现金流 - 资本支出
```

**预测要点**:
```
- 基于历史数据
- 考虑增长驱动因素
- 保守估计 (不要过于乐观)
- 通常预测 5-10 年
```

**例子**:
```
年份    1    2    3    4    5
FCF   100  120  144  173  207  (假设 20% 增长)
```

---

#### 第 2 步：确定折现率 ⭐⭐⭐⭐

**折现率 (r)**: 你要求的最低回报率

**常用方法**:
```
1. 无风险利率 + 风险溢价
   - 无风险利率：10 年期国债收益率 (约 3%)
   - 风险溢价：根据风险程度 (5%-10%)
   - 折现率：8%-13%

2. WACC (加权平均资本成本)
   - 更专业，考虑债务成本
   - 公式复杂，需要更多数据
```

**参考**:
```
- 低风险公司：8%-10%
- 中等风险：10%-12%
- 高风险：12%-15%
```

---

#### 第 3 步：计算终值 ⭐⭐⭐

**终值**: 预测期后的持续价值

**常用方法**:

##### 永续增长法
```
终值 = FCFn × (1+g) / (r-g)

其中:
- FCFn: 最后一年现金流
- g: 永续增长率 (通常 2%-5%)
- r: 折现率

注意：g 必须 < r，否则公式无意义
```

##### 退出倍数法
```
终值 = 最后一年 FCF × 退出 PE
```

---

#### 第 4 步：折现求和

```
将每年的 FCF 和终值折现到今天:

第 1 年现值 = 100 / (1.1)^1 = 90.9
第 2 年现值 = 120 / (1.1)^2 = 99.2
第 3 年现值 = 144 / (1.1)^3 = 108.2
第 4 年现值 = 173 / (1.1)^4 = 118.2
第 5 年现值 = 207 / (1.1)^5 = 128.5
终值现值 = 2500 / (1.1)^5 = 1552.3

公司价值 = 90.9 + 99.2 + 108.2 + 118.2 + 128.5 + 1552.3
        = 2097.3
```

---

#### 第 5 步：计算每股价值

```
每股价值 = 公司价值 / 总股本

例子:
公司价值 = 2097 亿
总股本 = 100 亿股
每股价值 = 20.97 元

当前股价 = 15 元 → 低估，可以买入
当前股价 = 25 元 → 高估，应该卖出
```

---

### DCF 的敏感性分析

**关键**: DCF 对假设非常敏感

**示例**:
```
| 折现率\增长率 | 8% | 10% | 12% |
|-------------|----|-----|-----|
| 2% | 高 | 中 | 低 |
| 3% | 很高 | 高 | 中 |
| 4% | 极高 | 很高 | 高 |
```

**建议**:
```
- 做多种假设 (保守/中性/乐观)
- 得到估值区间而非单一数字
- 只在价格远低于保守估值时买入
```

---

### DCF 的优缺点

#### ✅ 优点
```
- 理论上最严谨的估值方法
- 基于现金流，不易被操纵
- 考虑时间价值
- 适合稳定现金流的公司
```

#### ❌ 缺点
```
- 对假设非常敏感
- 预测未来很困难
- 不适合现金流不稳定的公司
- 计算复杂
- "精确的错误"风险
```

---

### DCF 适用场景

#### ✅ 适合
```
- 现金流稳定的公司
- 成熟行业龙头
- 消费、公用事业等
- 能够合理预测未来
```

#### ❌ 不适合
```
- 初创公司 (现金流为负)
- 周期性行业 (现金流波动大)
- 高成长公司 (难以预测)
- 困境反转公司
```

---

### DCF 实战建议

1. **保守假设**: 宁可低估不要高估
2. **多情景分析**: 保守/中性/乐观
3. **安全边际**: 价格低于保守估值 30%+ 才买
4. **定期更新**: 根据新信息调整假设
5. **结合其他方法**: 不要只用 DCF

---

## Q64: 如何设置止盈止损点？

### 答案

### 止盈止损的重要性

**目的**:
```
- 止盈：锁定利润，避免坐过山车
- 止损：控制亏损，避免大亏
- 纪律：克服人性弱点 (贪婪和恐惧)
```

---

### 止损策略

#### 1. 固定比例止损 ⭐⭐⭐

**方法**:
```
- 买入后下跌 X% 就卖出
- 常见设置：5%、8%、10%、15%
```

**优点**:
```
✅ 简单明确
✅ 纪律性强
✅ 避免大亏
```

**缺点**:
```
❌ 可能被洗盘震出
❌ 不考虑公司基本面
❌ 震荡市容易反复止损
```

**适用**:
```
- 短线交易
- 趋势交易
- 不确定性的投资
```

---

#### 2. 技术位止损 ⭐⭐⭐

**方法**:
```
- 跌破支撑位止损
- 跌破均线止损 (如 20 日、60 日线)
- 跌破趋势线止损
```

**优点**:
```
✅ 符合技术分析逻辑
✅ 避免正常波动被洗出
```

**缺点**:
```
❌ 需要技术分析能力
❌ 支撑位可能被击穿后反弹
```

**适用**:
```
- 技术分析交易者
- 趋势跟踪策略
```

---

#### 3. 基本面止损 ⭐⭐⭐⭐⭐

**方法**:
```
- 买入逻辑被证伪
- 公司基本面恶化
- 行业发生重大不利变化
- 发现更好的投资机会
```

**优点**:
```
✅ 基于价值投资
✅ 避免错过好公司
✅ 长期有效
```

**缺点**:
```
❌ 需要深入研究
❌ 判断可能滞后
❌ 短期可能承受较大亏损
```

**适用**:
```
- 价值投资
- 长期投资
- 深度研究的公司
```

---

#### 4. 时间止损 ⭐⭐

**方法**:
```
- 买入后 X 时间不涨就卖出
- 常见：3 个月、6 个月、1 年
```

**优点**:
```
✅ 提高资金效率
✅ 避免长期不涨
```

**缺点**:
```
❌ 可能卖在启动前
❌ 好公司需要时间
```

**适用**:
```
- 资金有限
- 追求效率
```

---

### 止盈策略

#### 1. 目标价位止盈 ⭐⭐⭐

**方法**:
```
- 设定目标价，达到就卖出
- 基于估值：如 PE 达到历史高位
- 基于涨幅：如翻倍就卖
```

**优点**:
```
✅ 锁定利润
✅ 避免贪婪
```

**缺点**:
```
❌ 可能卖飞大牛股
❌ 目标价难以设定
```

---

#### 2. 分批止盈 ⭐⭐⭐⭐⭐

**方法**:
```
- 涨 50% 卖 1/4
- 涨 100% 卖 1/4
- 涨 150% 卖 1/4
- 剩余长期持有
```

**优点**:
```
✅ 锁定部分利润
✅ 保留上涨空间
✅ 心态好
```

**缺点**:
```
❌ 收益不是最大化
```

**适用**:
```
- 大部分投资者
- 不确定能涨多少时
```

---

#### 3. 移动止盈 (追踪止损) ⭐⭐⭐⭐

**方法**:
```
- 从最高点回撤 X% 就卖出
- 常见：回撤 10%、15%、20%
```

**优点**:
```
✅ 让利润奔跑
✅ 锁定大部分利润
✅ 适合趋势行情
```

**缺点**:
```
❌ 震荡市可能反复
❌ 需要实时监控
```

**适用**:
```
- 趋势交易
- 大牛股持有
```

---

#### 4. 估值止盈 ⭐⭐⭐⭐⭐

**方法**:
```
- 估值过高时卖出
- PE 超过历史 80% 分位
- PEG > 2
- 市值明显高估
```

**优点**:
```
✅ 基于价值
✅ 避免高位接盘
✅ 长期有效
```

**缺点**:
```
❌ 高估可能更高估
❌ 需要估值能力
```

**适用**:
```
- 价值投资
- 长期投资
```

---

### 止盈止损设置建议

#### 不同投资风格的设置

| 风格 | 止损 | 止盈 |
|------|------|------|
| 短线交易 | 5%-8% | 15%-30% |
| 中线波段 | 10%-15% | 50%-100% |
| 长线投资 | 基本面恶化 | 估值过高 |
| 价值投资 | 逻辑证伪 | 分批 + 估值 |

---

#### 不同仓位的设置

```
重仓 (>30%): 严格止损 (8%-10%)
中仓 (10%-30%): 适中止损 (10%-15%)
轻仓 (<10%): 宽松止损 (15%-20%)
```

---

### 止盈止损的纪律

```
✅ 提前设定：买入前就想好
✅ 严格执行：到了点位就执行
✅ 不要移动：不要因为亏损而下调止损
✅ 记录复盘：总结每次止盈止损
```

---

### 不止损的情况

```
✅ 深度研究的好公司
✅ 买入逻辑未变
✅ 只是市场波动
✅ 长期投资 (5 年+)
✅ 有持续现金流 (如定投)
```

**但要注意**: 不止损≠不止亏，基本面恶化必须止损

## Q65: 仓位管理有哪些方法？

### 答案

### 仓位管理的重要性

**目的**:
```
- 控制风险
- 平滑波动
- 提高长期收益
- 保持良好心态
```

**核心**: 不要把所有鸡蛋放在一个篮子里

---

### 仓位管理方法

#### 1. 固定仓位法 ⭐⭐⭐

**方法**:
```
- 每只股票固定仓位 (如 10%)
- 总仓位固定 (如 80% 股票 +20% 现金)
```

**优点**:
```
✅ 简单易懂
✅ 分散风险
✅ 避免单只股票风险
```

**缺点**:
```
❌ 不够灵活
❌ 好公司不能多买
```

**适用**:
```
- 新手投资者
- 没时间研究的投资者
```

---

#### 2. 金字塔建仓法 ⭐⭐⭐⭐⭐

**方法**:
```
- 首次买入：计划仓位的 30%-50%
- 下跌加仓：每跌 X% 加仓一次
- 加仓金额递减：形成金字塔

例子 (计划买 10 万):
- 首次：买 3 万 (30%)
- 跌 10%: 买 2 万 (20%)
- 跌 20%: 买 1.5 万 (15%)
- 跌 30%: 买 1 万 (10%)
- 剩余：备用
```

**优点**:
```
✅ 摊薄成本
✅ 避免一次性买在高点
✅ 心态好
```

**缺点**:
```
❌ 可能错过快速上涨
❌ 需要预留资金
```

**适用**:
```
- 震荡市
- 不确定的底部
- 大部分情况
```

---

#### 3. 倒金字塔建仓法 ⭐⭐⭐

**方法**:
```
- 首次买入：计划仓位的 20%-30%
- 上涨加仓：每涨 X% 加仓一次
- 加仓金额递增

例子:
- 首次：买 2 万
- 涨 10%: 买 3 万
- 涨 20%: 买 5 万
```

**优点**:
```
✅ 顺势而为
✅ 确认趋势后重仓
```

**缺点**:
```
❌ 成本越来越高
❌ 可能买在顶部
```

**适用**:
```
- 趋势交易
- 强势股
- 右侧交易
```

---

#### 4. 凯利公式 ⭐⭐⭐

**公式**:
```
f* = (p × b - q) / b

其中:
- f*: 最优仓位比例
- p: 胜率
- q: 败率 (1-p)
- b: 盈亏比 (盈利/亏损)
```

**例子**:
```
胜率 60%，盈亏比 2:1
f* = (0.6 × 2 - 0.4) / 2 = 0.4 = 40%

建议仓位：40%
```

**优点**:
```
✅ 数学上最优
✅ 长期收益最大化
```

**缺点**:
```
❌ 胜率和盈亏比难以准确估计
❌ 波动大
❌ 实际中建议半凯利 (f*/2)
```

---

#### 5. 风险平价法 ⭐⭐⭐

**方法**:
```
- 按风险贡献分配仓位
- 高风险资产低仓位，低风险资产高仓位

例子:
- 股票 (高风险): 30%
- 债券 (中风险): 40%
- 现金 (低风险): 30%
```

**优点**:
```
✅ 风险均衡
✅ 适合资产配置
```

**缺点**:
```
❌ 计算复杂
❌ 需要风险数据
```

---

#### 6. 动态再平衡 ⭐⭐⭐⭐⭐

**方法**:
```
- 设定目标配置比例
- 定期 (如每季度) 检查
- 偏离过大时调整回目标

例子 (目标：股 60%+ 债 40%):
- 股票大涨后：股 70% + 债 30%
- 再平衡：卖出股票，买入债券
- 恢复：股 60% + 债 40%
```

**优点**:
```
✅ 自动高抛低吸
✅ 控制风险
✅ 纪律性强
```

**缺点**:
```
❌ 可能降低收益 (大牛市中)
❌ 需要定期操作
```

**适用**:
```
- 长期投资
- 资产配置
- 退休账户
```

---

### 仓位管理实战建议

#### 1. 总仓位控制
```
牛市：70%-100%
震荡市：50%-70%
熊市：20%-50%
极熊：0%-20% (或空仓)
```

#### 2. 单只股票仓位
```
- 单只上限：不超过 20%-30%
- 前五大持仓：不超过 60%
- 行业集中度：不超过 40%
```

#### 3. 根据确定性调整
```
高确定性 (深度研究): 15%-25%
中确定性 (一般研究): 5%-15%
低确定性 (尝试性): 1%-5%
```

#### 4. 根据市场环境调整
```
估值低 + 情绪差：高仓位
估值高 + 情绪好：低仓位
不确定：中等仓位
```

---

### 仓位管理禁忌

```
❌ 满仓单只股票 (风险太大)
❌ 借钱炒股 (杠杆危险)
❌ 频繁满仓空仓 (容易错)
❌ 亏损后加仓摊平 (可能越套越深)
❌ 盈利后盲目加仓 (可能高位接盘)
```

---

### 仓位管理检查清单

```
□ 单只股票是否超过上限？
□ 行业是否过于集中？
□ 总仓位是否匹配市场环境？
□ 是否有足够现金应对波动？
□ 是否需要再平衡？
□ 仓位是否与风险承受能力匹配？
```
