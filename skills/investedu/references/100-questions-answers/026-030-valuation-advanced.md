## Q26: 什么是市销率 (PS)？

### 答案

### PS 定义

**市销率 (Price-to-Sales Ratio)**

**公式**:
```
PS = 总市值 / 总营业收入
或
PS = 股价 / 每股销售收入
```

**简单理解**: 
> **PS = 你花多少钱买 1 元销售收入**

---

### PS 计算示例

```
某科技公司:
- 总市值：1000 亿元
- 年营业收入：100 亿元
- PS = 1000 / 100 = 10 倍

含义:
- 花 10 元买 1 元销售收入
```

---

### PS 适用场景

✅ **适合用 PS 估值**:
- 未盈利的成长公司
- 互联网公司
- 研发投入大的公司
- 周期性亏损公司

❌ **不适合用 PS 估值**:
- 盈利稳定的传统公司
- 重资产公司
- 收入波动大的公司

---

### PS 的优缺点

| 优点 | 缺点 |
|------|------|
| 亏损公司也能用 | 不考虑盈利能力 |
| 收入难造假 | 不考虑成本 |
| 适合成长股 | 不同行业差异大 |

---

### 各行业 PS 参考

| 行业 | 合理 PS 范围 |
|------|-------------|
| **传统制造** | 1-3 倍 |
| **消费** | 3-5 倍 |
| **互联网** | 5-15 倍 |
| **SaaS 软件** | 10-20 倍 |
| **生物科技** | 波动大 |

## Q27: 什么是股息率？

### 答案

### 股息率定义

**股息率 (Dividend Yield)**

**公式**:
```
股息率 = 每股股息 / 股价 × 100%
或
股息率 = 总分红 / 总市值 × 100%
```

**简单理解**: 
> **股息率 = 存款利率的股票版**

---

### 股息率计算示例

```
工商银行:
- 股价：5 元
- 每股分红：0.3 元
- 股息率 = 0.3 / 5 = 6%

含义:
- 相当于 6% 的年利率
- 比银行存款高很多
```

---

### 股息率的意义

| 股息率 | 含义 | 适合 |
|--------|------|------|
| **<1%** | 很低 | 成长股 |
| **1-3%** | 正常 | 大多数公司 |
| **3-5%** | 较高 | 价值股 |
| **>5%** | 很高 | 银行、公用事业 |
| **>10%** | 异常高 | 可能有陷阱 |

---

### 高股息率陷阱

❌ **警惕异常高股息**:
```
股息率>10% 可能原因:
- 股价暴跌 (基本面恶化)
- 一次性分红 (不可持续)
- 周期高点 (即将下滑)
- 财务造假
```

✅ **正确筛选高股息**:
- 连续 5 年分红
- 分红率稳定
- 盈利稳定
- 现金流好

---

### 股息率 vs 银行利率

| 指标 | 股息率 | 银行利率 |
|------|--------|----------|
| **收益** | 3-6% | 1-3% |
| **风险** | 中高 | 极低 |
| **流动性** | 高 | 高 |
| **税收** | 持股>1 年免税 | 利息税 20% |

---

### 股息投资策略

**高股息组合**:
- 银行股
- 公用事业
- 高速公路
- REITs

**优点**:
- 稳定现金流
- 复利再投资
- 抗通胀

**缺点**:
- 成长性低
- 股价波动

## Q28: 什么是 ROE？

### 答案

### ROE 定义

**净资产收益率 (Return on Equity)**

**公式**:
```
ROE = 净利润 / 净资产 × 100%
```

**简单理解**: 
> **ROE = 公司用股东的钱赚钱的能力**

---

### ROE 计算示例

```
贵州茅台:
- 年净利润：500 亿元
- 净资产：1500 亿元
- ROE = 500 / 1500 = 33.3%

含义:
- 股东每投入 100 元，赚 33.3 元
- 盈利能力非常强
```

---

### ROE 的意义

| ROE 范围 | 含义 | 评价 |
|----------|------|------|
| **<5%** | 很低 | 差 |
| **5-10%** | 较低 | 一般 |
| **10-15%** | 正常 | 良好 |
| **15-20%** | 较高 | 优秀 |
| **>20%** | 很高 | 非常优秀 |
| **>30%** | 极高 | 罕见 (茅台级别) |

---

### 巴菲特的 ROE 标准

> "我只投资 ROE 持续高于 15% 的公司。"

> "ROE 比 EPS 更重要。"

**巴菲特选股标准**:
- ROE 连续 5 年>15%
- 负债率低
- 盈利稳定

---

### ROE 的杜邦分析

```
ROE = 净利率 × 周转率 × 杠杆率

1. 净利率 = 净利润 / 收入
   (盈利能力)

2. 周转率 = 收入 / 资产
   (运营效率)

3. 杠杆率 = 资产 / 净资产
   (财务杠杆)
```

**高 ROE 的三种模式**:
1. **高利润型**: 茅台 (净利率高)
2. **高周转型**: 沃尔玛 (周转快)
3. **高杠杆型**: 银行 (杠杆高)

---

### ROE 注意事项

❌ **警惕**:
- ROE 突然升高 (可能是一次性收益)
- 高杠杆推高的 ROE
- 周期高点的 ROE

✅ **关注**:
- 连续 5-10 年稳定 ROE
- 低负债的高 ROE
- 行业领先的 ROE

## Q29: 什么是 DCF 估值法？

### 答案

### DCF 定义

**现金流折现法 (Discounted Cash Flow)**

**核心思想**: 
> **公司价值 = 未来所有现金流的现值之和**

**简单理解**: 
> **DCF = 把未来的钱折算成今天的价值**

---

### DCF 公式

```
公司价值 = Σ(第 t 年自由现金流 / (1+ 折现率)^t)

其中:
- t = 年份 (1, 2, 3...n)
- 折现率 = 期望回报率 (通常 8-12%)
```

---

### DCF 计算步骤

#### 步骤 1: 预测自由现金流
```
自由现金流 = 经营现金流 - 资本支出

预测未来 5-10 年:
- 第 1 年：100 亿
- 第 2 年：120 亿
- 第 3 年：144 亿
- ...
```

#### 步骤 2: 确定折现率
```
通常用 WACC (加权平均资本成本)
- 无风险利率：3%
- 风险溢价：5-7%
- 折现率：8-10%
```

#### 步骤 3: 计算现值
```
第 1 年现值 = 100 / (1.1)^1 = 90.9 亿
第 2 年现值 = 120 / (1.1)^2 = 99.2 亿
第 3 年现值 = 144 / (1.1)^3 = 108.2 亿
...
```

#### 步骤 4: 计算终值
```
终值 = 最后一年现金流 × (1+ 永续增长率) / (折现率 - 永续增长率)

假设永续增长率 2%:
终值 = 最后一年现金流 × 1.02 / (0.1 - 0.02)
```

#### 步骤 5: 加总求和
```
公司价值 = 所有年份现值 + 终值现值
```

---

### DCF 的优缺点

| 优点 | 缺点 |
|------|------|
| 理论上最正确 | 预测很困难 |
| 考虑长期价值 | 对假设敏感 |
| 不受市场情绪影响 | 计算复杂 |

---

### DCF 适用场景

✅ **适合**:
- 现金流稳定的公司
- 成熟企业
- 可以预测的公司

❌ **不适合**:
- 初创公司
- 现金流不稳定的公司
- 周期股

---

### 简化 DCF

**快速估算**:
```
合理市值 = 自由现金流 / (折现率 - 增长率)

例子:
- 自由现金流：100 亿
- 折现率：10%
- 增长率：5%
- 合理市值 = 100 / (0.1 - 0.05) = 2000 亿
```

## Q30: 如何判断股票是低估还是高估？

### 答案

### 判断方法总览

```
1. PE 法
2. PB 法
3. PEG 法
4. DCF 法
5. 历史比较法
6. 同行比较法
7. 股息率法
```

---

### 1. PE 法

```
低估：PE < 行业平均 或 PE < 历史中位数
高估：PE > 行业平均 或 PE > 历史中位数

例子:
- 某股票 PE=15 倍
- 行业平均 PE=25 倍
- 结论：可能低估
```

---

### 2. PB 法

```
低估：PB < 1 或 PB < 历史低位
高估：PB > 5 或 PB > 历史高位

适合：银行、重资产行业
```

---

### 3. PEG 法

```
PEG = PE / 净利润增长率

PEG < 1: 低估
PEG = 1: 合理
PEG > 1: 高估

例子:
- PE=30 倍
- 增长率=40%
- PEG = 30/40 = 0.75 (低估)
```

---

### 4. DCF 法

```
计算内在价值:
- 内在价值 > 当前股价：低估
- 内在价值 < 当前股价：高估
```

---

### 5. 历史比较法

```
查看股票历史 PE/PB:
- 当前 PE 处于历史 10% 分位：低估
- 当前 PE 处于历史 90% 分位：高估
```

---

### 6. 同行比较法

```
与同行业公司比较:
- PE 低于同行：可能低估
- PE 高于同行：可能高估

注意：要考虑公司质量差异
```

---

### 7. 股息率法

```
股息率 > 历史平均：可能低估
股息率 < 历史平均：可能高估

适合：高分红公司
```

---

### 综合判断框架

```
低估信号 (满足 3 个以上):
□ PE 低于行业平均
□ PB 低于历史中位数
□ PEG < 1
□ 股息率高于历史
□ 股价低于内在价值
□ 市场情绪悲观

高估信号 (满足 3 个以上):
□ PE 高于行业平均
□ PB 高于历史中位数
□ PEG > 2
□ 股息率低于历史
□ 股价高于内在价值
□ 市场情绪狂热
```

---

### 巴菲特判断法

```
1. 这家公司我能看懂吗？(能力圈)
2. 公司有护城河吗？(竞争优势)
3. 管理层可靠吗？(诚信)
4. 价格有安全边际吗？(低估)

4 个都是"是"→ 可以投资
```

---

### 安全边际

**定义**: 用大幅低于价值的价格买入

**建议**:
```
- 保守投资者：5 折买入 (安全边际 50%)
- 稳健投资者：7 折买入 (安全边际 30%)
- 激进投资者：8 折买入 (安全边际 20%)
```

---

### 新手建议

1. **从简单开始**: 先用 PE、PB 判断
2. **多方法验证**: 不要只看一个指标
3. **留安全边际**: 宁可错过，不要买贵
4. **持续学习**: 估值是艺术 + 科学
