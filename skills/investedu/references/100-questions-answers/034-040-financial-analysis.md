## Q34: 什么是毛利率？

### 答案

### 毛利率定义

**公式**:
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
或
毛利率 = 毛利 / 营业收入 × 100%
```

**简单理解**: 
> **毛利率 = 产品本身的赚钱能力**
> 
> **毛利率 = 1 元收入中有多少是毛利**

---

### 毛利率计算示例

```
贵州茅台:
- 营业收入：1000 亿元
- 营业成本：100 亿元
- 毛利 = 1000 - 100 = 900 亿元
- 毛利率 = 900 / 1000 = 90%

含义:
- 每卖 100 元酒，成本只有 10 元
- 毛利 90 元
- 毛利率极高!
```

---

### 毛利率的意义

| 毛利率 | 含义 | 行业 |
|--------|------|------|
| **>80%** | 极高 | 白酒、软件 |
| **50-80%** | 很高 | 医药、互联网 |
| **30-50%** | 较高 | 消费、科技 |
| **20-30%** | 正常 | 制造、服务 |
| **10-20%** | 较低 | 零售、代工 |
| **<10%** | 很低 | 贸易、批发 |

---

### 各行业毛利率参考

| 行业 | 典型毛利率 |
|------|------------|
| **白酒** | 80-95% |
| **生物制药** | 80-90% |
| **软件/SaaS** | 70-90% |
| **医疗器械** | 60-80% |
| **食品饮料** | 40-60% |
| **家电** | 30-40% |
| **汽车** | 15-25% |
| **建筑** | 10-15% |
| **贸易** | 3-8% |

---

### 毛利率分析要点

#### 1. 看趋势
```
毛利率上升 → 产品竞争力增强/成本下降
毛利率下降 → 竞争加剧/成本上升
```

#### 2. 看同行
```
毛利率高于同行 → 竞争优势
毛利率低于同行 → 竞争劣势
```

#### 3. 看稳定性
```
毛利率稳定 → 商业模式好
毛利率波动大 → 周期性/竞争激烈
```

---

### 高毛利率的秘诀

```
1. 品牌溢价 (茅台、苹果)
2. 技术壁垒 (华为、高通)
3. 垄断地位 (腾讯微信)
4. 转换成本高 (微软 Office)
5. 稀缺资源 (片仔癀)
```

## Q35: 什么是净利率？

### 答案

### 净利率定义

**公式**:
```
净利率 = 净利润 / 营业收入 × 100%
```

**简单理解**: 
> **净利率 = 公司最终赚钱能力**
> 
> **净利率 = 1 元收入中有多少是净利润**

---

### 净利率 vs 毛利率

```
营业收入：100 元
  - 营业成本：60 元
  ─────────────
  毛利：40 元 (毛利率 40%)
  
  - 销售费用：10 元
  - 管理费用：10 元
  - 研发费用：5 元
  - 财务费用：5 元
  - 所得税：4 元
  ─────────────
  净利润：6 元 (净利率 6%)
```

**关键**: 毛利率高不代表净利率高

---

### 净利率的意义

| 净利率 | 含义 | 评价 |
|--------|------|------|
| **>20%** | 很高 | 优秀 |
| **10-20%** | 较高 | 良好 |
| **5-10%** | 正常 | 一般 |
| **<5%** | 较低 | 较差 |
| **<0%** | 亏损 | 危险 |

---

### 各行业净利率参考

| 行业 | 典型净利率 |
|------|------------|
| **白酒** | 30-50% |
| **互联网** | 20-30% |
| **生物制药** | 15-25% |
| **食品饮料** | 10-20% |
| **家电** | 5-10% |
| **汽车** | 5-8% |
| **银行** | 25-35% |
| **建筑** | 2-4% |
| **零售** | 1-3% |

---

### 净利率分析要点

#### 1. 看趋势
```
净利率上升 → 管理效率提高/规模效应
净利率下降 → 费用增加/竞争加剧
```

#### 2. 看费用控制
```
销售费用率 = 销售费用 / 收入
管理费用率 = 管理费用 / 收入

费用率下降 → 管理改善
```

#### 3. 看非经常性损益
```
扣非净利润更真实:
- 卖资产收益 (一次性)
- 政府补助 (不稳定)
- 投资收益 (波动大)
```

## Q36: 什么是应收账款？

### 答案

### 应收账款定义

**定义**: 应收账款是公司销售商品或提供服务后，客户欠公司的钱。

**简单理解**: 
> **应收账款 = 白条 = 别人欠公司的钱**

---

### 应收账款分析

#### 1. 应收账款/收入比率
```
比率 = 应收账款 / 营业收入

- <20%: 健康
- 20-30%: 正常
- >30%: 警惕
- >50%: 危险
```

#### 2. 应收账款周转率
```
周转率 = 营业收入 / 平均应收账款

周转率高 → 回款快
周转率低 → 回款慢
```

#### 3. 应收账款周转天数
```
天数 = 365 / 周转率

- <30 天：很好
- 30-60 天：正常
- 60-90 天：警惕
- >90 天：危险
```

---

### 应收账款风险

| 风险 | 说明 |
|------|------|
| **坏账风险** | 客户不还钱 |
| **资金占用** | 影响现金流 |
| **利润虚增** | 确认收入但没收到钱 |
| **财务造假** | 虚构应收账款 |

---

### 警惕信号

❌ **危险信号**:
- 应收账款增速 > 收入增速
- 周转天数持续上升
- 大客户欠款集中
- 账龄长的应收账款多
- 频繁变更坏账计提政策

---

## Q37: 什么是存货？

### 答案

### 存货定义

**定义**: 存货是公司准备销售的商品、生产中的产品、原材料等。

**简单理解**: 
> **存货 = 库存 = 还没卖出去的东西**

---

### 存货分类

| 类型 | 说明 |
|------|------|
| **原材料** | 准备生产用的材料 |
| **在产品** | 正在生产中的产品 |
| **库存商品** | 生产好还没卖的 |
| **周转材料** | 包装物、低值易耗品 |

---

### 存货分析

#### 1. 存货/收入比率
```
比率 = 存货 / 营业收入

- <20%: 健康
- 20-30%: 正常
- >30%: 警惕
- >50%: 危险
```

#### 2. 存货周转率
```
周转率 = 营业成本 / 平均存货

周转率高 → 卖得快
周转率低 → 卖得慢
```

#### 3. 存货周转天数
```
天数 = 365 / 周转率

- <30 天：很好 (快消品)
- 30-60 天：正常
- 60-90 天：警惕
- >90 天：危险 (可能积压)
```

---

### 存货风险

| 风险 | 说明 |
|------|------|
| **跌价风险** | 产品过时/降价 |
| **积压风险** | 卖不出去 |
| **变质风险** | 食品、药品过期 |
| **财务造假** | 虚构存货 |

---

### 警惕信号

❌ **危险信号**:
- 存货增速 > 收入增速
- 周转天数持续上升
- 存货跌价准备少
- 库存商品占比高
- 产品更新换代快

---

## Q38: 什么是商誉？

### 答案

### 商誉定义

**定义**: 商誉是公司收购其他公司时，支付价格超过被收购公司净资产的部分。

**简单理解**: 
> **商誉 = 收购溢价 = 买贵了的部分**

---

### 商誉计算

```
商誉 = 收购价格 - 被收购公司净资产

例子:
A 公司收购 B 公司:
- B 公司净资产：10 亿元
- A 公司支付：30 亿元
- 商誉 = 30 - 10 = 20 亿元
```

**含义**: A 公司认为 B 公司值 30 亿，但账面只有 10 亿，多付的 20 亿就是商誉。

---

### 商誉的本质

```
商誉代表:
- 品牌价值
- 客户关系
- 技术优势
- 团队价值
- 协同效应

这些在账面上没有体现，但确实有价值
```

---

### 商誉风险

#### 商誉减值
```
当被收购公司业绩不达预期:
- 需要计提商誉减值
- 直接冲减利润
- 可能导致巨额亏损
```

**例子**:
```
2018 年 A 股商誉减值潮:
- 多家公司计提巨额商誉减值
- 有的公司因此亏损几十亿
- 股价大跌
```

---

### 商誉分析

#### 1. 商誉/净资产比率
```
比率 = 商誉 / 净资产

- <10%: 安全
- 10-20%: 注意
- 20-30%: 警惕
- >30%: 危险
```

#### 2. 商誉/总资产比率
```
比率 = 商誉 / 总资产

- <5%: 安全
- >10%: 警惕
```

---

### 警惕信号

❌ **危险信号**:
- 商誉/净资产 > 30%
- 频繁收购 (尤其是高溢价)
- 被收购公司业绩承诺未完成
- 商誉减值准备不足
- 收购跨界 (不懂的行业)

---

### 巴菲特谈商誉

> "我们宁愿要好的商誉，不要好的商誉会计处理。"

> "真正的好公司不需要靠收购来增长。"

## Q39: 如何识别财务造假？

### 答案

### 财务造假常见手段

#### 1. 虚增收入
```
手段:
- 虚构销售合同
- 提前确认收入
- 关联交易输送利益
- 循环交易

识别:
- 应收账款异常增长
- 经营现金流与利润背离
- 客户集中度高
- 毛利率远高于同行
```

#### 2. 虚减成本
```
手段:
- 少结转成本
- 延迟确认费用
- 资本化费用

识别:
- 毛利率异常高
- 存货异常增长
- 在建工程长期不转固
```

#### 3. 虚增资产
```
手段:
- 虚构货币资金
- 虚增存货
- 虚增固定资产

识别:
- 货币资金多但利息少
- 存货周转率异常低
- 固定资产产出比低
```

#### 4. 隐瞒负债
```
手段:
- 表外负债
- 未披露担保
- 关联方借款

识别:
- 其他应付款异常
- 利息支出与借款不匹配
- 对外担保多
```

---

### 财务造假预警信号

#### 红旗信号 (高度警惕)

| 信号 | 说明 |
|------|------|
| **存贷双高** | 货币资金多 + 有息负债多 |
| **利润现金流背离** | 利润高但经营现金流差 |
| **应收账款暴增** | 增速远超收入增速 |
| **存货暴增** | 周转率大幅下降 |
| **毛利率异常** | 远高于同行且无合理解释 |
| **频繁换会计师** | 尤其是年报前更换 |
| **大股东高比例质押** | >70% 质押 |
| **频繁并购** | 尤其是跨界高溢价 |
| **关联交易多** | 利益输送嫌疑 |
| **被监管问询** | 交易所/证监会问询 |

---

### 康美药业造假案例

```
造假手段:
- 虚增货币资金 887 亿元
- 虚增营业收入 291 亿元
- 虚增利润 40 亿元

识别信号:
- 存贷双高 (货币资金 300 亿 + 借款 200 亿)
- 利息收入与货币资金不匹配
- 应收账款周转慢
- 大股东高比例质押

结果:
- 2019 年曝光
- 退市
- 相关人员入刑
```

---

### 财务造假识别方法

#### 1. 三表勾稽
```
利润表利润 → 现金流量表经营现金流 → 资产负债表未分配利润

如果不匹配，可能有问题
```

#### 2. 同业比较
```
与同行业公司比较:
- 毛利率
- 净利率
- 周转率
- 负债率

异常高或低都要警惕
```

#### 3. 历史比较
```
与自己历史比较:
- 毛利率突然上升
- 费用率突然下降
- 应收账款突然增加

突变要警惕
```

#### 4. 常识判断
```
- 毛利率远超同行且无合理解释
- 业绩增长但行业下行
- 研发费用低但技术领先

不符合常识的要警惕
```

---

### 防造假口诀

```
一看收入现金比 (经营现金流/收入)
二看应收存货比 (应收账款 + 存货)/收入
三看毛利同行比 (毛利率 vs 同行)
四看商誉净资产比 (商誉/净资产)
五看大股东质押 (质押比例)
六看会计师意见 (审计意见类型)
```

## Q40: 什么是杜邦分析？

### 答案

### 杜邦分析定义

**定义**: 杜邦分析法是将 ROE 分解为多个指标，分析公司盈利能力的方法。

**起源**: 1920 年代杜邦公司发明

**简单理解**: 
> **杜邦分析 = ROE 的 X 光片**
> 
> **看透 ROE 背后的驱动因素**

---

### 杜邦分析公式

#### 经典三因素分解

```
ROE = 净利率 × 周转率 × 杠杆率

其中:
1. 净利率 = 净利润 / 收入
   (盈利能力)

2. 周转率 = 收入 / 总资产
   (运营效率)

3. 杠杆率 = 总资产 / 净资产
   (财务杠杆)
```

---

### 杜邦分析图解

```
                    ROE (净资产收益率)
                         │
        ┌────────────────┼────────────────┐
        │                │                │
    净利率           周转率           杠杆率
        │                │                │
   盈利能力         运营效率         财务杠杆
        │                │                │
   ┌────┴────┐      ┌────┴────┐      ┌────┴────┐
   │         │      │         │      │         │
 净利润    收入   收入    总资产  总资产   净资产
```

---

### 三种盈利模式

#### 1. 高利润模式 (茅台型)
```
特点:
- 净利率高 (>30%)
- 周转率一般
- 杠杆率低

代表:
- 贵州茅台 (净利率 50%+)
- 片仔癀
- 高端白酒

优势:
- 盈利能力强
- 风险低
```

#### 2. 高周转模式 (沃尔玛型)
```
特点:
- 净利率低 (2-5%)
- 周转率高
- 杠杆率一般

代表:
- 沃尔玛
- 亚马逊
- 京东

优势:
- 规模效应
- 现金流好
```

#### 3. 高杠杆模式 (银行型)
```
特点:
- 净利率一般 (20-30%)
- 周转率低
- 杠杆率高 (10 倍+)

代表:
- 银行
- 保险
- 房地产

风险:
- 财务风险高
- 受政策影响大
```

---

### 杜邦分析应用

#### 1. 找差距
```
公司 A ROE=15%
公司 B ROE=25%

分解:
A: 净利率 10% × 周转率 0.8 × 杠杆率 1.5 = 12%
B: 净利率 15% × 周转率 1.0 × 杠杆率 1.7 = 25.5%

差距主要在净利率和周转率
```

#### 2. 看趋势
```
ROE 从 20% 降到 15%:

去年：20% × 1.0 × 1.5 = 30%
今年：15% × 1.0 × 1.5 = 22.5%

原因：净利率下降 (竞争加剧)
```

#### 3. 选股票
```
优选:
- 高净利率 + 稳定周转 + 低杠杆 (最安全)
- 中净利率 + 高周转 + 中杠杆 (也不错)

避免:
- 低净利率 + 低周转 + 高杠杆 (危险)
```

---

### 杜邦分析局限

❌ **局限性**:
- 只看财务指标
- 不考虑非财务因素
- 历史数据不能预测未来

✅ **正确使用**:
- 结合行业特点
- 结合竞争优势
- 结合管理层质量

---

### 巴菲特与杜邦分析

> "我们寻找高 ROE、低负债的公司。"

巴菲特喜欢的公司特征:
- ROE > 15%
- 净利率高
- 负债率低
- 不需要高杠杆就能获得高 ROE
