## Q31: 什么是资产负债表？

### 答案

### 资产负债表定义

**定义**: 资产负债表是反映公司在某一特定日期财务状况的报表。

**简单理解**: 
> **资产负债表 = 公司的家底清单**
> 
> **照片 = 某一时刻的快照**

---

### 资产负债表结构

```
资产负债表

资产 (钱去哪了)          负债和所有者权益 (钱从哪来)
─────────────────────────────────────────────────────
流动资产                  流动负债
  货币资金                  短期借款
  应收账款                  应付账款
  存货                      应付职工薪酬
  预付款项                  应交税费
                            其他应付款
非流动资产                非流动负债
  固定资产                  长期借款
  在建工程                  应付债券
  无形资产                  递延所得税
  商誉
  长期股权投资
                            所有者权益
                              实收资本
                              资本公积
                              盈余公积
                              未分配利润
─────────────────────────────────────────────────────
资产总计 = 负债总计 + 所有者权益
```

---

### 核心公式

```
资产 = 负债 + 所有者权益

这个公式永远平衡!
```

**含义**:
- 左边：公司有什么 (资产)
- 右边：钱从哪来 (借的 + 股东的)

---

### 重要科目解读

#### 资产方 (重要科目)

| 科目 | 说明 | 关注点 |
|------|------|--------|
| **货币资金** | 现金 + 银行存款 | 越多越好，但别太多 |
| **应收账款** | 别人欠公司的钱 | 越少越好，周转快 |
| **存货** | 库存商品 | 适中，别积压 |
| **固定资产** | 厂房设备等 | 重资产 vs 轻资产 |
| **商誉** | 收购溢价 | 越低越好，有减值风险 |

#### 负债方 (重要科目)

| 科目 | 说明 | 关注点 |
|------|------|--------|
| **短期借款** | 1 年内要还的借款 | 越少越好 |
| **应付账款** | 欠供应商的钱 | 适度，太多可能拖欠 |
| **长期借款** | 1 年后还的借款 | 看负债率 |
| **应付债券** | 发行的债券 | 利息负担 |

#### 所有者权益

| 科目 | 说明 |
|------|------|
| **实收资本** | 股东投入的本金 |
| **未分配利润** | 历年赚的钱累积 |

---

### 如何分析资产负债表？

#### 1. 看负债率
```
资产负债率 = 总负债 / 总资产

- <40%: 安全
- 40-60%: 正常
- >60%: 风险较高
- >80%: 危险
```

#### 2. 看流动性
```
流动比率 = 流动资产 / 流动负债

- >2: 流动性好
- 1-2: 正常
- <1: 流动性紧张
```

#### 3. 看资产质量
```
好资产：货币资金、应收账款 (能收回)
差资产：商誉、长期待摊费用 (可能减值)
```

---

### 资产负债表健康标准

| 指标 | 健康标准 |
|------|----------|
| 资产负债率 | <50% |
| 流动比率 | >1.5 |
| 货币资金/总资产 | >10% |
| 应收账款/收入 | <20% |
| 存货/收入 | <30% |
| 商誉/净资产 | <20% |

## Q32: 什么是利润表？

### 答案

### 利润表定义

**定义**: 利润表是反映公司在一定会计期间经营成果的报表。

**简单理解**: 
> **利润表 = 公司的赚钱能力**
> 
> **视频 = 一段时间的表现**

---

### 利润表结构

```
利润表

一、营业收入
    减：营业成本
        税金及附加
        销售费用
        管理费用
        研发费用
        财务费用
二、营业利润
    加：营业外收入
    减：营业外支出
三、利润总额
    减：所得税费用
四、净利润
    其中：归属于母公司股东的净利润
          少数股东损益
五、每股收益 (EPS)
```

---

### 重要科目解读

| 科目 | 说明 | 关注点 |
|------|------|--------|
| **营业收入** | 公司卖产品/服务的收入 | 增长是否稳定 |
| **营业成本** | 生产产品的直接成本 | 成本控制 |
| **销售费用** | 广告、销售人员工资 | 占比是否合理 |
| **管理费用** | 管理人员工资、办公费 | 是否节约 |
| **研发费用** | 研发投入 | 科技公司看这个 |
| **财务费用** | 利息支出 | 负债高则费用高 |
| **净利润** | 最终赚的钱 | 最重要指标 |

---

### 利润表分析要点

#### 1. 看收入增长
```
收入增长率 = (今年收入 - 去年收入) / 去年收入

- >20%: 高增长
- 10-20%: 稳定增长
- 0-10%: 低速增长
- <0%: 衰退
```

#### 2. 看利润率
```
毛利率 = (收入 - 成本) / 收入
净利率 = 净利润 / 收入

毛利率:
- >50%: 很高 (茅台 90%+)
- 30-50%: 较高
- 20-30%: 正常
- <20%: 较低

净利率:
- >20%: 优秀
- 10-20%: 良好
- 5-10%: 一般
- <5%: 较低
```

#### 3. 看利润质量
```
好利润：主营业务利润 (可持续)
差利润：营业外收入、投资收益 (不可持续)
```

---

### 利润表健康标准

| 指标 | 健康标准 |
|------|----------|
| 收入增长率 | >10% |
| 毛利率 | >30% (因行业而异) |
| 净利率 | >10% |
| 净利润增长率 | >10% |
| 扣非净利润占比 | >80% |

## Q33: 什么是现金流量表？

### 答案

### 现金流量表定义

**定义**: 现金流量表是反映公司在一定会计期间现金流入和流出的报表。

**简单理解**: 
> **现金流量表 = 公司的现金流水账**
> 
> **现金 = 公司的血液**

---

### 现金流量表结构

```
现金流量表

一、经营活动产生的现金流量
    销售商品、提供劳务收到的现金
    收到的税费返还
    收到其他与经营活动有关的现金
    购买商品、接受劳务支付的现金
    支付给职工以及为职工支付的现金
    支付的各项税费
    支付其他与经营活动有关的现金
    ─────────────────────────────
    经营活动产生的现金流量净额 ← 最重要!

二、投资活动产生的现金流量
    收回投资收到的现金
    取得投资收益收到的现金
    处置固定资产收回的现金
    购建固定资产支付的现金
    投资支付的现金
    ─────────────────────────────
    投资活动产生的现金流量净额

三、筹资活动产生的现金流量
    吸收投资收到的现金
    取得借款收到的现金
    偿还债务支付的现金
    分配股利、利润支付的现金
    ─────────────────────────────
    筹资活动产生的现金流量净额

四、现金及现金等价物净增加额
```

---

### 三类活动解读

#### 1. 经营活动 (最重要)
```
正数 (+)：经营赚钱，现金流入
负数 (-)：经营亏钱，现金流出

理想状态：持续为正，且增长
```

#### 2. 投资活动
```
正数 (+)：变卖资产，收回投资
负数 (-)：购买资产，对外投资

扩张期：通常为负 (正常)
收缩期：通常为正
```

#### 3. 筹资活动
```
正数 (+)：借钱、融资
负数 (-)：还钱、分红

成熟公司：通常为负 (还钱 + 分红)
成长公司：通常为正 (融资扩张)
```

---

### 现金流量表分析

#### 1. 经营现金流 vs 净利润
```
经营现金流净额 / 净利润

- >1: 利润质量好 (真赚钱)
- =1: 正常
- <1: 利润质量差 (纸面富贵)
- <0.5: 警惕 (可能造假)
```

#### 2. 自由现金流
```
自由现金流 = 经营现金流 - 资本支出

自由现金流>0: 公司有钱
自由现金流<0: 公司缺钱

巴菲特最看重这个指标!
```

#### 3. 现金流类型组合

| 经营 | 投资 | 筹资 | 公司状态 |
|------|------|------|----------|
| + | - | + | 成长期 (正常) |
| + | - | - | 成熟期 (理想) |
| + | + | - | 收缩期 |
| - | - | + | 危险期 (警惕) |
| - | + | + | 衰退期 (危险) |

---

### 现金流量表健康标准

| 指标 | 健康标准 |
|------|----------|
| 经营现金流净额 | >0 且增长 |
| 经营现金流/净利润 | >1 |
| 自由现金流 | >0 |
| 现金/总资产 | >10% |

---

### 三表关系总结

| 报表 | 反映 | 特点 |
|------|------|------|
| **资产负债表** | 家底 | 时点数据 |
| **利润表** | 赚钱能力 | 时期数据 |
| **现金流量表** | 现金流水 | 时期数据 |

**关系**:
```
利润表解释：钱是怎么赚的
现金流量表解释：钱是真的吗
资产负债表解释：钱去哪了
```
