## Q11: 什么是股票？股票的本质是什么？

### 答案

### 股票的定义

**定义**: 股票是股份有限公司发行的所有权凭证，代表持有者对公司的部分所有权。

**简单理解**: 
> **买股票 = 买公司的一部分**

---

### 股票的本质

#### 1. 所有权凭证
```
持有股票 = 公司股东
- 有权分享公司利润 (分红)
- 有权参与重大决策 (投票)
- 有权分配剩余资产 (清算时)
```

#### 2. 不是债务凭证
```
股票 vs 债券:
- 股票：所有权，不保本
- 债券：债权，到期还本付息
```

#### 3. 可以交易
```
股票可以在证券市场买卖:
- 价格由市场决定
- 可能上涨也可能下跌
- 流动性好 (T+1 交易)
```

---

### 股东的权利

| 权利 | 说明 |
|------|------|
| **分红权** | 分享公司利润 |
| **投票权** | 参与重大决策 |
| **知情权** | 查阅公司财报 |
| **转让权** | 自由买卖股票 |
| **优先认购权** | 增发时优先购买 |

---

### 股票的收益来源

#### 1. 资本利得 (价差)
```
低买高卖:
- 买入：10 元
- 卖出：15 元
- 收益：5 元 (50%)
```

#### 2. 股息分红
```
公司盈利分红:
- 持有 1000 股
- 每股分红 0.5 元
- 分红收入：500 元
```

#### 3. 送股/转增
```
10 送 5:
- 原有 1000 股
- 送股后：1500 股
- 总价值不变，股数增加
```

---

### 股票的特点

| 特点 | 说明 |
|------|------|
| **收益性** | 可能获得分红和价差 |
| **风险性** | 可能亏损，甚至本金全失 |
| **流动性** | 可以随时买卖 |
| **永久性** | 没有到期日 |
| **参与性** | 可以参与公司决策 |

---

### 股票类型

#### 按股东权利分:
- **普通股**: 有投票权，分红不固定
- **优先股**: 无投票权，优先分红

#### 按上市地点分:
- **A 股**: 中国大陆，人民币交易
- **B 股**: 中国大陆，外币交易
- **H 股**: 香港上市，中资企业
- **美股**: 美国上市

#### 按公司规模分:
- **大盘股**: 市值>1000 亿
- **中盘股**: 市值 100-1000 亿
- **小盘股**: 市值<100 亿

---

### 巴菲特谈股票

> "买股票就是买公司的一部分。"

> "如果你不愿意持有一只股票 10 年，那就不要考虑拥有它 10 分钟。"

| 项目 | 说明 |

| **交易货币** | 上海用美元，深圳用港币 |

| **代码** | 沪市 900 开头，深市 200 开头 |

| **交易货币** | 港币 |
| **投资者** | 全球投资者 |
| **公司类型** | 注册在内地，在香港上市 |

**例子**: 中国移动、工商银行、中石油

**特点**: 估值通常比 A 股低，分红率高

---

| **涨跌幅** | 无限制 |

**特点**: 全球最大股市，科技公司多

## Q13: 什么是 IPO？IPO 流程是怎样的？

### 答案

### IPO 定义

**IPO**: Initial Public Offering (首次公开募股)

**定义**: 私人公司第一次向公众发行股票并上市交易。

**简单理解**: 
> **IPO = 公司第一次公开卖股票**

---

### 为什么要 IPO？

#### 1. 融资
```
上市可以:
- 募集大量资金
- 用于扩张、研发、还债
```

#### 2. 提升知名度
```
上市公司:
- 品牌曝光
- 客户信任
- 吸引人才
```

#### 3. 股东退出
```
早期投资者:
- 风投/PE 可以退出
- 创始团队可以套现
```

#### 4. 并购工具
```
上市公司:
- 可以用股票收购其他公司
- 更容易扩张
```

---

### IPO 流程

#### 阶段 1: 准备期 (6-12 个月)
```
1. 聘请中介机构
   - 保荐人 (券商)
   - 会计师
   - 律师

2. 股份制改造
   - 有限责任公司→股份有限公司

3. 财务规范
   - 审计财报
   - 规范税务
```

#### 阶段 2: 申报期 (3-6 个月)
```
4. 提交 IPO 申请
   - 向证监会/交易所提交材料

5. 审核问询
   - 监管审核
   - 回复问询

6. 过会
   - 上市委员会审议
```

#### 阶段 3: 发行期 (1-2 个月)
```
7. 路演
   - 向机构投资者推介

8. 询价
   - 确定发行价格

9. 申购
   - 网上/网下申购

10. 挂牌上市
    - 正式交易
```

---

### A 股 IPO 条件 (主板)

| 条件 | 要求 |
|------|------|
| **持续经营** | 3 年以上 |
| **净利润** | 最近 3 年均为正，累计>3000 万 |
| **营收** | 最近 3 年累计>3 亿 |
| **股本** | 发行前>3000 万股 |
| **无形资产** | 占比≤20% |

---

### IPO 投资风险

| 风险 | 说明 |
|------|------|
| **破发风险** | 上市后跌破发行价 |
| **估值过高** | 发行价太贵 |
| **业绩变脸** | 上市后业绩下滑 |
| **流动性风险** | 新股可能交易不活跃 |

---

### 新股申购技巧

1. **坚持申购**: 中签率低，但坚持有机会
2. **选择时段**: 上午 10:00-11:00，下午 14:00-15:00
3. **分散账户**: 家人多个账户提高概率
4. **研究基本面**: 好公司值得申购

## Q14: 什么是交易所？中国有哪些交易所？

### 答案

### 交易所定义

**定义**: 证券交易所是为股票、债券等证券交易提供场所和设施的机构。

**简单理解**: 
> **交易所 = 股票买卖的市场**

---

### 交易所的职能

| 职能 | 说明 |
|------|------|
| **提供交易场所** | 买卖双方交易的平台 |
| **制定交易规则** | 交易时间、涨跌幅等 |
| **组织监督交易** | 确保公平交易 |
| **信息披露** | 公布公司信息 |
| **监管会员** | 管理券商 |

## Q15: 什么是主板、创业板、科创板？

### 答案

### 主板

**定位**: 大型成熟企业

| 项目 | 说明 |
|------|------|
| **企业类型** | 大型、成熟、盈利稳定 |
| **上市条件** | 最严格 |
| **风险等级** | 相对较低 |
| **投资者门槛** | 无 |
| **涨跌幅限制** | 10% |
| **股票代码** | 沪市 600/601, 深市 000 |

**代表公司**: 贵州茅台、工商银行、中国石油

---

### 创业板

**定位**: 成长型创新创业企业

| 项目 | 说明 |
|------|------|
| **企业类型** | 成长型、创新型 |
| **上市条件** | 较宽松 |
| **风险等级** | 较高 |
| **投资者门槛** | 2 年经验 +10 万资金 |
| **涨跌幅限制** | 20% |
| **股票代码** | 300 开头 |

**代表公司**: 宁德时代、迈瑞医疗、东方财富

---

### 科创板

**定位**: 硬科技创新企业

| 项目 | 说明 |
|------|------|
| **企业类型** | 科技创新、核心技术 |
| **上市条件** | 允许未盈利企业 |
| **风险等级** | 高 |
| **投资者门槛** | 2 年经验 +50 万资金 |
| **涨跌幅限制** | 20% |
| **股票代码** | 688 开头 |

**代表公司**: 中芯国际、金山办公、澜起科技

---

### 北交所

**定位**: 创新型中小企业

| 项目 | 说明 |
|------|------|
| **企业类型** | 专精特新中小企业 |
| **上市条件** | 相对宽松 |
| **风险等级** | 高 |
| **投资者门槛** | 2 年经验 +50 万资金 |
| **涨跌幅限制** | 30% |
| **股票代码** | 8/4/9 开头 |

---

### 各板块对比

| 板块 | 门槛 | 风险 | 成长性 | 适合投资者 |
|------|------|------|--------|------------|
| **主板** | 无 | 低 | 中 | 新手 ⭐⭐⭐⭐⭐ |
| **创业板** | 10 万 | 中高 | 高 | 有经验⭐⭐⭐⭐ |

---

### 新手建议

1. **从主板开始**: 风险相对较低
2. **学习后再投创业板**: 需要 2 年经验
3. **科创板谨慎**: 门槛高，风险大
4. **指数基金**: 可以投资各板块指数
