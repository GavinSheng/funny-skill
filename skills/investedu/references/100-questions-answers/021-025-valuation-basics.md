## Q21: 什么是 K 线？K 线如何看？

### 答案

### K 线定义

**K 线**: 又称蜡烛图、日本蜡烛图，是记录股票价格变动的图表。

**起源**: 18 世纪日本米商本间宗久发明，用于记录大米价格。

**简单理解**: 
> **K 线 = 股票价格的可视化记录**

---

### K 线的四个价格

每根 K 线包含 4 个关键价格:

```
1. 开盘价 (Open): 当天第一笔成交价格
2. 收盘价 (Close): 当天最后一笔成交价格
3. 最高价 (High): 当天最高成交价格
4. 最低价 (Low): 当天最低成交价格
```

---

### K 线结构

#### 阳线 (上涨)
```
     │←上影线→│
     ┌─────────┐ ← 收盘价
     │         │
     │  实体   │
     │         │
     └─────────┘ ← 开盘价
     │←下影线→│
```

#### 阴线 (下跌)
```
     │←上影线→│
     ┌─────────┐ ← 开盘价
     │         │
     │  实体   │
     │         │
     └─────────┘ ← 收盘价
     │←下影线→│
```

---

### K 线颜色

| 市场 | 阳线颜色 | 阴线颜色 |
|------|----------|----------|
| **A 股** | 红色 (涨) | 绿色 (跌) |
| **美股** | 绿色 (涨) | 红色 (跌) |
| **国际** | 白色/空心 (涨) | 黑色/实心 (跌) |

**记忆口诀**: 
> **A 股：红涨绿跌**

---

### K 线周期

| 周期 | 说明 | 用途 |
|------|------|------|
| **分时 K 线** | 1 分钟、5 分钟、15 分钟 | 短线交易 |
| **日 K 线** | 每天一根 | 最常用 |
| **周 K 线** | 每周一根 | 中线趋势 |
| **月 K 线** | 每月一根 | 长线趋势 |
| **年 K 线** | 每年一根 | 超长期 |

---

### 常见 K 线形态

#### 单根 K 线

| 形态 | 特征 | 含义 |
|------|------|------|
| **大阳线** | 实体长，无影线 | 强烈看涨 |
| **大阴线** | 实体长，无影线 | 强烈看跌 |
| **十字星** | 开盘=收盘 | 多空平衡 |
| **锤头线** | 下影线长 | 可能见底 |
| **吊颈线** | 上影线长 | 可能见顶 |

#### 双 K 线组合

| 形态 | 特征 | 含义 |
|------|------|------|
| **吞没形态** | 后根包住前根 | 趋势反转 |
| **孕线** | 后根在前根内 | 趋势犹豫 |
| **乌云盖顶** | 阴穿阳一半 | 看跌 |
| **曙光初现** | 阳穿阴一半 | 看涨 |

#### 多 K 线组合

| 形态 | 特征 | 含义 |
|------|------|------|
| **早晨之星** | 阴 + 星 + 阳 | 底部反转 |
| **黄昏之星** | 阳 + 星 + 阴 | 顶部反转 |
| **红三兵** | 连续 3 阳 | 强烈看涨 |
| **黑三鸦** | 连续 3 阴 | 强烈看跌 |

---

### K 线的局限性

❌ **K 线不是万能的**:
- 只能反映历史价格
- 不能预测未来
- 可能被操纵

✅ **正确使用方法**:
- 结合成交量
- 结合基本面
- 结合其他指标

## Q22: 什么是阳线、阴线？

### 答案

### 阳线

**定义**: 收盘价高于开盘价的 K 线。

**特征**:
```
- 收盘价 > 开盘价
- A 股显示为红色
- 表示当天股价上涨
```

**阳线结构**:
```
     最高价
       │
   ┌───┴───┐ ← 收盘价
   │       │
   │ 红色  │
   │ 实体  │
   │       │
   └───┬───┘ ← 开盘价
       │
     最低价
```

**阳线含义**:
- 买方力量强
- 股价上涨
- 多头占优

---

### 阴线

**定义**: 收盘价低于开盘价的 K 线。

**特征**:
```
- 收盘价 < 开盘价
- A 股显示为绿色
- 表示当天股价下跌
```

**阴线结构**:
```
     最高价
       │
   ┌───┬───┐ ← 开盘价
   │   │   │
   │ 绿色  │
   │ 实体  │
   │       │
   └───┴───┘ ← 收盘价
       │
     最低价
```

**阴线含义**:
- 卖方力量强
- 股价下跌
- 空头占优

---

### 阳线 vs 阴线对比

| 特征 | 阳线 | 阴线 |
|------|------|------|
| **收盘 vs 开盘** | 收盘>开盘 | 收盘<开盘 |
| **A 股颜色** | 红色 | 绿色 |
| **市场情绪** | 乐观 | 悲观 |
| **多空力量** | 多头强 | 空头强 |

---

### 特殊形态

#### 十字星 (开盘=收盘)
```
     │
   ───┼───
     │
```
- 多空平衡
- 可能变盘

#### 一字线 (涨停/跌停)
```
涨停：────  (无上下影线)
跌停：────  (无上下影线)
```
- 极度强势/弱势
- 一边倒

## Q23: 什么是大盘指数？

### 答案

### 大盘指数定义

**定义**: 大盘指数是反映整个股票市场或某一类股票价格变动的指标。

**简单理解**: 
> **大盘指数 = 股市的温度计**

---

### 主要大盘指数

#### A 股主要指数

| 指数 | 代码 | 代表意义 |
|------|------|----------|
| **上证指数** | 000001 | 沪市全部股票 |
| **深证成指** | 399001 | 深市代表性股票 |
| **创业板指** | 399006 | 创业板股票 |
| **沪深 300** | 000300 | A 股 300 只龙头 |
| **中证 500** | 000905 | 中盘 500 只股票 |
| **上证 50** | 000016 | 沪市 50 只超大盘 |

---

#### 港股主要指数

| 指数 | 代表意义 |
|------|----------|
| **恒生指数** | 港股最具代表性 |
| **恒生国企** | 香港上市的中资企业 |
| **恒生科技** | 科技股指数 |

---

#### 美股主要指数

| 指数 | 代表意义 |
|------|----------|
| **道琼斯** | 30 只蓝筹股 |
| **标普 500** | 500 只大盘股 |
| **纳斯达克** | 科技股为主 |

---

### 指数计算方法

#### 加权方式

**1. 市值加权 (主流)**
```
指数 = Σ(股价 × 总股本) / 基期

特点:
- 大公司权重高
- 茅台涨 1% 影响大
- 小公司影响小
```

**2. 等权重**
```
每只股票权重相同
特点:
- 小公司影响大
- 更分散
```

**3. 价格加权**
```
按股价加权
特点:
- 高价股权重大
- 道琼斯使用
```

---

### 指数的作用

| 作用 | 说明 |
|------|------|
| **市场晴雨表** | 反映市场整体表现 |
| **投资基准** | 衡量投资收益 |
| **投资工具** | 指数基金跟踪 |
| **情绪指标** | 判断市场热度 |

---

### 如何使用指数？

#### 1. 判断市场趋势
```
指数上涨 → 市场好 → 可以投资
指数下跌 → 市场差 → 谨慎投资
```

#### 2. 比较投资收益
```
你的收益 vs 指数收益:
- 跑赢指数 → 投资能力强
- 跑输指数 → 不如买指数基金
```

#### 3. 定投指数基金
```
最简单投资方式:
- 沪深 300 指数基金
- 标普 500 指数基金
- 长期定投
```

---

### 新手建议

1. **关注沪深 300**: 代表 A 股龙头
2. **不要只看上证指数**: 不够全面
3. **指数基金是好选择**: 简单有效
4. **长期看指数向上**: 不要怕短期波动

## Q24: 什么是市盈率 (PE)？

### 答案

### PE 定义

**市盈率 (Price-to-Earnings Ratio)**

**公式**:
```
PE = 股价 / 每股收益 (EPS)
或
PE = 总市值 / 总净利润
```

**简单理解**: 
> **PE = 回本年限**
> 
> **PE=10 表示 10 年回本**

---

### PE 计算示例

```
贵州茅台:
- 股价：1500 元
- 每股收益：50 元
- PE = 1500 / 50 = 30 倍

含义:
- 按当前盈利，30 年回本
- 投资者愿意为 1 元盈利支付 30 元
```

---

### PE 的类型

#### 1. 静态 PE
```
使用去年年报数据
优点：数据确定
缺点：可能过时
```

#### 2. 动态 PE
```
使用预测今年数据
优点：反映未来
缺点：预测可能不准
```

#### 3. 滚动 PE (TTM)
```
使用最近 4 个季度数据
优点：最新数据
缺点：受季节影响

⭐ 最推荐使用 TTM PE
```

---

### PE 的含义

| PE 范围 | 含义 | 适合 |
|---------|------|------|
| **<10 倍** | 低估 | 价值投资 |
| **10-20 倍** | 合理 | 大多数公司 |
| **20-30 倍** | 偏高 | 成长股 |
| **>30 倍** | 高估 | 高成长股 |
| **>100 倍** | 极高 | 风险大 |

---

### PE 的局限性

❌ **不能只看 PE**:
- 不同行业 PE 不同
- 周期股 PE 失真
- 亏损公司无 PE
- 一次性收益影响

✅ **正确使用方法**:
- 同行业比较
- 历史 PE 比较
- 结合成长性 (PEG)
- 看扣非后 PE

---

### 各行业 PE 参考

| 行业 | 合理 PE 范围 |
|------|-------------|
| **银行** | 5-10 倍 |
| **地产** | 5-15 倍 |
| **消费** | 20-30 倍 |
| **医药** | 25-40 倍 |
| **科技** | 30-50 倍 |
| **周期** | 波动大 |

---

### 巴菲特谈 PE

> "宁可要模糊的正确，不要精确的错误。"

> "PE 低不一定好，要看公司质量。"

## Q25: 什么是市净率 (PB)？

### 答案

### PB 定义

**市净率 (Price-to-Book Ratio)**

**公式**:
```
PB = 股价 / 每股净资产
或
PB = 总市值 / 总净资产
```

**简单理解**: 
> **PB = 你花多少钱买 1 元净资产**

---

### PB 计算示例

```
工商银行:
- 股价：5 元
- 每股净资产：6 元
- PB = 5 / 6 = 0.83 倍

含义:
- 花 0.83 元买 1 元净资产
- 股价低于净资产 (破净)
```

---

### PB 的含义

| PB 范围 | 含义 | 适合 |
|---------|------|------|
| **<1 倍** | 破净，低估 | 银行、周期 |
| **1-3 倍** | 合理 | 大多数公司 |
| **3-5 倍** | 偏高 | 成长股 |
| **>5 倍** | 高估 | 轻资产公司 |

---

### PB 适用场景

✅ **适合用 PB 估值**:
- 银行、保险
- 重资产行业
- 周期股
- 净资产稳定的公司

❌ **不适合用 PB 估值**:
- 轻资产公司 (互联网)
- 服务业
- 高成长公司
- 净资产为负的公司

---

### PE vs PB 对比

| 指标 | 适用 | 不适用 |
|------|------|--------|
| **PE** | 盈利稳定的公司 | 亏损、周期股 |
| **PB** | 重资产公司 | 轻资产、高成长 |

**最佳实践**: 两个指标结合看

---

### 破净股

**定义**: PB<1，股价低于净资产

**特点**:
- 理论上很便宜
- 但可能有陷阱
- 需要分析原因

**破净原因**:
- 行业不景气
- 资产质量差
- 盈利能力弱
- 市场情绪差
