## Q6: 什么是风险？投资有哪些风险？

### 答案

---

### 什么是风险？

**定义**: 投资风险是指实际收益偏离预期收益的可能性，包括亏损的可能。

**简单理解**: 
> **风险 = 不确定性 = 可能亏钱**

**关键点**:
- 风险≠一定会亏钱
- 风险=可能亏钱的概率
- 高风险≠高收益 (只是可能)

---

### 投资的主要风险类型

#### 一、市场风险 (系统性风险) 📉

**定义**: 整个市场下跌带来的风险，无法通过分散投资消除

**表现**:
- 股灾
- 经济危机
- 战争
- 疫情

**例子**:
- 2008 年全球金融危机：美股跌 57%
- 2015 年 A 股股灾：上证指数跌 45%
- 2020 年疫情：全球股市跌 30%+

**应对**: 
- 资产配置 (股票 + 债券)
- 长期持有
- 定期定投

---

#### 二、个股风险 (非系统性风险) 🏢

**定义**: 单个公司特有的风险，可以通过分散投资降低

**表现**:
- 公司业绩下滑
- 管理层问题
- 产品失败
- 财务造假

**例子**:
- 乐视网：从 1700 亿市值到退市
- 康美药业：财务造假 300 亿
- 长生生物：疫苗事件退市

**应对**:
- 分散投资 (至少 10 只股票)
- 投资指数基金
- 深入研究公司

---

#### 三、利率风险 📊

**定义**: 利率变化导致投资价格波动的风险

**影响**:
- 利率上升 → 债券价格下跌
- 利率上升 → 股票估值下降
- 利率下降 → 债券价格上涨

**例子**:
```
持有 10 年期国债，利率 3%:
- 央行加息到 4%
- 你的债券价格下跌约 10%
```

**应对**:
- 缩短债券久期
- 浮动利率债券
- 资产配置

---

#### 四、通胀风险 💸

**定义**: 通胀导致购买力下降的风险

**影响**:
- 现金购买力下降
- 固定收益资产实际收益为负

**例子**:
```
存款利率 2%，通胀 3%:
- 实际收益：-1%
- 钱越存越少
```

**应对**:
- 投资股票、房产
- 配置抗通胀资产
- 避免大量现金

---

#### 五、流动性风险 💧

**定义**: 无法及时变现或变现时损失较大的风险

**表现**:
- 想卖卖不掉
- 必须大幅降价才能卖出

**例子**:
- 房产：可能需要几个月才能卖出
- 小盘股：成交量低，难以大额卖出
- 私募产品：有锁定期，无法提前赎回

**应对**:
- 保留应急资金 (3-6 个月)
- 投资流动性好的资产
- 不要全部投入长期资产

---

#### 六、信用风险 (违约风险) ⚠️

**定义**: 借款人无法按时还本付息的风险

**影响对象**:
- 债券投资
- P2P 理财
- 信托产品

**例子**:
- 某房企债券违约
- P2P 平台跑路
- 理财产品无法兑付

**应对**:
- 选择高信用评级债券
- 避免高息诱惑
- 分散投资

---

#### 七、汇率风险 💱

**定义**: 汇率波动导致投资收益变化的风险

**影响**:
- 投资海外资产
- 持有外币资产

**例子**:
```
投资美股:
- 股票涨 10%
- 人民币升值 5%
- 实际收益：约 5%
```

**应对**:
- 汇率对冲
- 分散多国投资
- 长期持有淡化汇率影响

---

#### 八、政策风险 🏛️

**定义**: 政策变化导致投资损失的风险

**表现**:
- 行业监管加强
- 税收政策变化
- 外资限制

**例子**:
- 2021 年教培行业：双减政策
- 2020 年互联网：反垄断
- 房地产：限购限贷

**应对**:
- 关注政策导向
- 避免政策敏感行业
- 分散行业配置

---

### 风险等级分类

| 产品 | 风险等级 | 最大回撤 | 适合人群 |
|------|----------|----------|----------|
| **存款/货币基金** | R1 (极低) | <1% | 所有人 |
| **国债** | R1 (极低) | <2% | 保守型 |
| **债券基金** | R2 (低) | 5-10% | 保守/稳健 |
| **混合基金** | R3 (中) | 10-20% | 稳健型 |
| **指数基金** | R3 (中) | 20-30% | 稳健/进取 |
| **股票基金** | R4 (中高) | 30-50% | 进取型 |
| **个股** | R5 (高) | 50-100% | 专业投资者 |
| **期货/期权** | R6 (极高) | >100% | 专业机构 |

### 风险与收益的关系

```
风险与收益成正比，但不是绝对的:

高风险 ≠ 高收益 (只是可能)
低风险 ≠ 低收益 (有时也有机会)

关键:
- 承担值得的风险
- 避免无谓的风险
- 在风险可控前提下追求收益
```

---

### 风险评估方法

#### 1. 最大回撤

```
最大回撤 = (最高价 - 最低价) / 最高价

例子:
- 股票最高 100 元
- 最低跌到 50 元
- 最大回撤：50%
```

#### 2. 波动率 (标准差)

```
波动率越大，风险越高

- 货币基金：波动率<1%
- 债券基金：波动率 3-5%
- 指数基金：波动率 15-20%
- 个股：波动率 30-50%
```

#### 3. 夏普比率

```
夏普比率 = (收益 - 无风险利率) / 波动率

- >1: 较好
- >2: 很好
- <0: 不如存银行
```

### 如何管理投资风险？

#### 1. 资产配置 ⭐ 最重要

```
不要把鸡蛋放在一个篮子里:
- 股票：50%
- 债券：30%
- 现金：10%
- 黄金：10%
```

#### 2. 分散投资

```
- 至少 10 只股票
- 或投资指数基金
- 跨行业、跨市场
```

#### 3. 定期定投

```
- 分批买入
- 降低择时风险
- 平摊成本
```

#### 4. 长期持有

```
- 短期波动大
- 长期趋势向上
- 时间平滑风险
```

#### 5. 设置止损

```
- 单只股票亏损>20% 考虑止损
- 组合回撤>15% 重新评估
```

#### 6. 持续学习

```
- 了解投资产品
- 学习风险管理
- 提高识别能力
```

---

### 风险承受能力评估

**问自己几个问题**:

1. 你能接受的最大亏损是多少？
   - 10% 以内 → 保守型
   - 10-20% → 稳健型
   - 20-30% → 进取型
   - 30% 以上 → 激进型

2. 你的投资期限是多久？
   - <1 年 → 低风险
   - 1-3 年 → 中风险
   - >5 年 → 可以高风险

3. 这笔钱急用吗？
   - 急用 → 不要投资高风险
   - 不急用 → 可以承受波动

---

### 给新手的建议

1. **了解自己的风险承受能力**: 不要投资超出承受范围的产品
2. **从低风险开始**: 货币基金→债券基金→指数基金→股票
3. **分散投资**: 不要 All in 单一资产
4. **长期思维**: 短期波动正常，长期向上
5. **持续学习**: 提高风险识别能力
6. **保留应急资金**: 3-6 个月生活费
