## L1 - 基础阶段 (0-3 个月)

### 模块 1.1: 投资基础概念

**学习目标**: 理解投资的基本概念和原理

| 知识点 | 内容 | 难度 |
|--------|------|------|
| 什么是投资 | 投资 vs 投机 vs 赌博 | ⭐ |
| 资产类别 | 股票、债券、现金、房地产、大宗商品 | ⭐ |
| 风险与收益 | 风险收益权衡、风险类型 | ⭐⭐ |
| 复利奇迹 | 复利计算、72 法则、时间价值 | ⭐⭐ |
| 通货膨胀 | 通胀影响、实际收益率 | ⭐ |

**推荐资源**:
- 📚 《小狗钱钱》- 博多·舍费尔
- 📚 《富爸爸穷爸爸》- 罗伯特·清崎
- 🎬 B 站：投资入门基础视频

---

### 模块 1.2: 金融市场概览

**学习目标**: 了解各类金融市场的基本运作

| 知识点 | 内容 | 难度 |
|--------|------|------|
| 股票市场 | 交易所、IPO、二级市场 | ⭐⭐ |
| 债券市场 | 国债、企业债、收益率 | ⭐⭐ |
| 基金市场 | 公募、私募、ETF | ⭐⭐ |
| 衍生品市场 | 期货、期权基础 | ⭐⭐⭐ |
| 外汇市场 | 汇率、外汇交易 | ⭐⭐ |
| 大宗商品 | 黄金、原油、农产品 | ⭐⭐ |

**推荐资源**:
- 📚 《金融市场与金融机构》- 米什金
- 🌐 Investopedia 网站

---

### 模块 1.3: 投资工具与操作

**学习目标**: 掌握基本投资工具的使用

| 知识点 | 内容 | 难度 |
|--------|------|------|
| 证券账户 | 开户流程、账户类型 | ⭐ |
| 交易软件 | 下单、撤单、查询 | ⭐ |
| 订单类型 | 市价单、限价单、止损单 | ⭐⭐ |
| 交易费用 | 佣金、印花税、过户费 | ⭐ |
| 交易时间 | 开盘、收盘、休市 | ⭐ |

**实践任务**:

- 完成 10 笔模拟交易

## L2 - 进阶阶段 (3-12 个月)

### 模块 2.1: 宏观经济分析

**学习目标**: 理解宏观经济对投资的影响

| 知识点 | 内容 | 难度 |
|--------|------|------|
| GDP 分析 | GDP 构成、增长率、实际 vs 名义 | ⭐⭐⭐ |
| 通货膨胀 | CPI、PPI、核心通胀 | ⭐⭐⭐ |
| 利率与货币 | 央行政策、利率决定、货币供应 | ⭐⭐⭐ |
| 财政政策 | 政府支出、税收、赤字 | ⭐⭐ |
| 经济周期 | 复苏、繁荣、衰退、萧条 | ⭐⭐⭐ |
| 国际收支 | 汇率、贸易、资本流动 | ⭐⭐⭐ |

**推荐资源**:
- 📚 《宏观经济学》- 曼昆
- 📚 《逃不开的经济周期》- 拉斯·特维德
- 📰 央行报告、统计局数据

---

| 竞争格局 | 市场集中度、竞争策略 | ⭐⭐⭐ |
| 政策分析 | 产业政策、监管政策 | ⭐⭐⭐ |
| 行业驱动 | 需求驱动、供给驱动、技术驱动 | ⭐⭐⭐ |

**推荐资源**:
- 📚 《竞争战略》- 迈克尔·波特

### 模块 2.3: 财务报表分析

## L3 - 专业阶段 (1-3 年)

### 模块 3.1: 深入估值建模

**学习目标**: 能够搭建完整的估值模型

| 知识点 | 内容 | 难度 |
|--------|------|------|
| DCF 详细建模 | 收入预测、成本预测、资本支出 | ⭐⭐⭐⭐ |
| WACC 计算 | 权益成本、债务成本、资本结构 | ⭐⭐⭐⭐ |
| 终值计算 | 永续增长法、退出倍数法 | ⭐⭐⭐⭐ |
| 敏感性分析 | 关键变量敏感性、情景分析 | ⭐⭐⭐⭐ |
| 实物期权 | 期权思维在估值中的应用 | ⭐⭐⭐⭐⭐ |
| 并购估值 | 协同效应、控制权溢价 | ⭐⭐⭐⭐ |

**实践任务**:
- 完成 5 份完整 DCF 估值报告
- 搭建可复用的估值模板

---

### 模块 3.2: 衍生品定价与策略

**学习目标**: 理解衍生品定价和对冲策略

| 知识点 | 内容 | 难度 |
|--------|------|------|
| 期货定价 | 持有成本模型、基差 | ⭐⭐⭐⭐ |
| 期权定价 | Black-Scholes 模型、二叉树 | ⭐⭐⭐⭐⭐ |
| 希腊字母 | Delta、Gamma、Theta、Vega | ⭐⭐⭐⭐ |
| 期权策略 | 保护性看跌、备兑开仓、跨式 | ⭐⭐⭐⭐ |
| 对冲策略 | 套期保值、风险对冲 | ⭐⭐⭐⭐ |
| 结构化产品 | 可转债、雪球、收益凭证 | ⭐⭐⭐⭐ |

**推荐资源**:
- 📚 《期权期货与其他衍生品》- 赫尔
- 📚 《期权策略》- 麦克米伦

---

**推荐资源**:

|------|------|

| 市场专家 | A 股/港股/美股/新兴市场 |
| 产品专家 | 股票/债券/衍生品/另类 |

### 模块 4.2: 投资框架构建

## 学习路径建议

### 路径 1: 价值投资方向

```
L1 基础 → L2 财报分析 → L2 估值基础 → L3 深入估值 → L4 实战
```

### 路径 2: 宏观策略方向

```
L1 基础 → L2 宏观经济 → L3 资产配置 → L4 实战
```

### 路径 3: 量化投资方向

```
L1 基础 → L2 统计基础 → L3 量化模型 → L3 回测 → L4 实战
```

### 路径 4: 行业研究方向

```
L1 基础 → L2 行业分析 → L2 财报分析 → L3 深度研究 → L4 专家
```
