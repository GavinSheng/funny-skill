# Canon EOS R3

## 基本规格
- **传感器**: 24.1MP 全画幅堆栈式CMOS
- **处理器**: DIGIC X
- **连拍速度**: 30fps 电子快门, 12fps 机械快门
- **视频**: 6K 60p RAW, 4K 120p
- **对焦系统**: Dual Pixel CMOS AF II, 眼部/头部/动物/车辆追踪

## 核心特色功能
### 1. 电子取景器 (EVF)
- 576万点 OLED EVF
- 120fps 刷新率
- 无黑屏连拍显示

### 2. 防抖系统
- 机身5轴防抖，最高8档补偿
- 与RF镜头协同防抖可达8.5档

### 3. 专业操控
- 竖拍手柄一体化设计
- 双CFexpress Type B卡槽
- 耐候密封机身

## 常用操作指南
### 设置眼部对焦
1. MENU → [AF] → [Subject to detect] → [People]
2. [Eye detection] → [On]
3. [AF method] → [Face+Tracking]

### 6K RAW视频录制
1. MENU → [Movie rec. size] → [6K RAW]
2. 确保使用CFexpress Type B卡
3. 注意录制时间限制和散热

### 高速连拍设置
1. 驱动模式选择高速连拍+
2. 使用电子快门获得30fps
3. 建议使用UHS-II SD卡或CFexpress卡

## 适用场景
- **体育摄影**: 30fps连拍 + 优秀追焦
- **野生动物**: 动物眼部追踪 + 高速连拍  
- **新闻摄影**: 耐候机身 + 双卡备份
- **视频制作**: 6K RAW内录 + 专业音频输入
