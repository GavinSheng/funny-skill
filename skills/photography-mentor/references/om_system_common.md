# OM System (Olympus) Brand Common Operations

## Overview
OM System (formerly Olympus) is known for its Micro Four Thirds system cameras, offering excellent image stabilization and compact form factors.

## Menu Navigation
- **Main Menu**: Access via MENU button
- **Quick Menu**: Press Q button for frequently used settings
- **Super Control Panel**: Customizable quick access panel

## Key Features
- **5-axis Image Stabilization**: Industry-leading in-body stabilization
- **Live Composite**: Unique long exposure mode for light painting
- **Pro Capture**: Buffer recording before full shutter press
- **High Res Shot**: Pixel-shift high resolution mode (on supported models)

## Common Settings Access
- **ISO**: Quick menu or Super Control Panel
- **White Balance**: Quick menu → WB
- **Focus Mode**: Fn button or Quick menu
- **Drive Mode**: Quick menu → Drive

## Button Customization
Most buttons can be customized through:
Menu → Setup → Button/Dial Setting → Button Function

## Typical Models
- OM-1 Mark II
- OM-5
