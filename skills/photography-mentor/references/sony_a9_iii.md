# Sony Alpha A9 III 差异化特性

## 与Sony通用操作的主要差异

### 全局快门革命
- **24MP全画幅**: 全局快门CMOS传感器
- **无果冻效应**: 高速运动拍摄无变形
- **超高速闪光同步**: 1/80000s闪光同步速度（传统机械快门限制为1/250s）
- **完全静音**: 无机械震动和声音

### 极速连拍性能
- **120fps连拍**: 全分辨率RAW连拍
- **无黑屏取景**: 连拍期间EVF持续显示
- **预拍摄功能**: 记录按下快门前1秒的图像
- **缓冲深度**: 显著增加的连拍缓冲容量

### AI智能对焦
- **深度学习AF**: 新一代AI主体识别算法
- **体育场景优化**: 专为体育摄影优化的追踪
- **车辆识别**: 赛车、摩托车专用识别模式
- **低光对焦**: -3EV扩展对焦范围

### 视频能力
- **6K超采样**: 4K 60p来自6K超采样
- **S-Log3**: 改进的动态范围表现
- **ProRes RAW**: 外录Apple ProRes RAW/HQ
- **散热优化**: 专业级散热系统

### 专业工作流
- **双CFexpress**: 双CFexpress Type A卡槽
- **USB供电**: 支持USB-C PD供电
- **FTP传输**: 内置无线FTP文件传输
- **PC遥控**: 增强的电脑远程控制

## 操作要点
- **全局快门模式**: 在菜单中专门设置，注意电池消耗增加
- **120fps连拍**: 需要CFexpress Type A卡，SD卡无法支持
- **预拍摄功能**: 在自定义键中分配，适合不可预测的瞬间
- **视频录制**: 6K超采样需要良好散热环境，建议准备备用相机
