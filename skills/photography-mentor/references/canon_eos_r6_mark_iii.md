# Canon EOS R6 Mark III 差异化特性

## 与Canon通用操作的主要差异

### 主要升级点
- **传感器**: 24.2MP 全画幅背照式CMOS（相比R6的20.1MP）
- **处理器**: DIGIC X 双处理器（性能提升约30%）
- **连拍速度**: 电子快门40fps，机械快门12fps
- **视频能力**: 6K超采样4K 60p，支持10-bit 4:2:2内录

### 对焦系统改进
- **深度学习AF**: 新的人眼/动物眼/车辆识别算法
- **低光对焦**: -6.5EV（比R6的-6EV略有提升）
- **主体识别**: 新增飞机、火车识别模式

### 视频功能增强
- **无裁切4K**: 相比R6 Mark II的轻微裁切，R6 III实现真正的无裁切4K
- **散热优化**: 改进的散热系统，延长录制时间
- **ProRes RAW**: 通过HDMI输出支持外录ProRes RAW

### 操作体验优化
- **电子取景器**: 3.69M点 OLED EVF，120fps刷新率
- **双卡槽**: 双CFexpress Type B卡槽
- **USB供电**: 支持USB-C PD供电和充电
- **电池续航**: 改进的电池管理系统

### 专业工作流
- **FTP传输**: 内置无线FTP文件传输
- **PC遥控**: 增强的电脑远程控制功能
- **元数据嵌入**: 可嵌入版权和联系信息
- **防尘防滴**: 专业级防护设计

## 操作要点
- **40fps连拍**: 需要CFexpress Type B卡，确保足够写入速度
- **6K超采样**: 在视频菜单中选择6K超采样4K选项
- **AI对焦**: 在对焦菜单中启用深度学习AF和具体主体识别
- **双卡槽**: 建议双CFexpress用于RAW+视频备份
- **散热管理**: 长时间4K 60p录制时注意温度监控
