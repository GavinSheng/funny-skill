# Fujifilm X-T5 Camera Guide

## Overview
The Fujifilm X-T5 is a flagship APS-C mirrorless camera featuring:
- 40.2MP X-Trans CMOS 5 HR sensor
- X-Processor 5 image processor
- 5-axis in-body image stabilization (up to 7 stops)
- 160MP Pixel Shift Multi-Shot
- 6.2K/30p video recording

## Key Features and Operations

### Physical Controls
- **Shutter Speed Dial**: Mechanical dial on top plate (A for auto)
- **ISO Dial**: Mechanical dial on top plate (A for auto)
- **Exposure Compensation Dial**: ±3EV with C position for extended range
- **Front Command Dial**: Default for aperture control
- **Rear Command Dial**: Default for exposure compensation

### Menu System Navigation
- **Q Menu**: Quick access to frequently used settings
- **Main Menu**: Organized into Shooting, Playback, and Setup tabs
- **My Menu**: Customizable favorites menu

### Film Simulation Modes
Fujifilm's signature feature with 19 film simulations:
- **PROVIA/Standard**: Balanced, natural colors
- **Velvia/Vivid**: High saturation and contrast
- **ASTIA/Soft**: Soft skin tones, lower contrast
- **Classic Chrome**: Subdued colors, popular for street photography
- **ACROS**: Monochrome with fine grain simulation

### Autofocus System
- **AF Mode**: Single Point, Zone, Wide/Tracking
- **Face/Eye Detection**: Human, Animal, Bird detection
- **Custom AF Settings**: Adjustable tracking sensitivity and speed

### Video Capabilities
- **6.2K/30p**: Highest quality video mode
- **4K/60p**: Standard high-quality option
- **F-Log2**: Flat profile for maximum dynamic range in post
- **Internal Recording**: 10-bit 4:2:2 color sampling

## Common Operations

### Setting Up for Still Photography
1. Set ISO dial to desired value or A
2. Set shutter speed dial to desired value or A  
3. Use front command dial to adjust aperture
4. Use rear command dial for exposure compensation
5. Press Q button to access quick menu for white balance, film simulation, etc.

### Setting Up for Video
1. Switch drive dial to Movie mode
2. Access Q menu and select video settings
3. Choose resolution/frame rate (6.2K/30p, 4K/60p, etc.)
4. Select F-Log2 for maximum dynamic range
5. Set audio levels via menu if using external mic

### Using Film Simulations Creatively
- **Street Photography**: Classic Chrome or ACROS
- **Landscape**: Velvia or PROVIA  
- **Portraits**: ASTIA or Classic Negative
- **Low Light**: Eterna or Pro Neg. Hi

## Troubleshooting Common Issues

### Battery Life Optimization
- Enable Power Save mode in setup menu
- Reduce EVF refresh rate to 60fps instead of 100fps
- Turn off Bluetooth when not needed
- Use mechanical dials instead of electronic controls

### Overheating Prevention (Video)
- Avoid direct sunlight on camera body
- Use external recorder for long sessions
- Enable heat dispersion setting in menu
- Take breaks between long recording sessions

### Focus Accuracy Issues
- Ensure correct AF mode for subject (Single vs Continuous)
- Adjust AF tracking sensitivity for moving subjects
- Use focus peaking for manual focus confirmation
- Calibrate lens if consistently front/back focusing
