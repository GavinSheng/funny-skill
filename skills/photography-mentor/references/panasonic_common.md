# Panasonic Camera Common Operations

## Brand Overview
Panasonic is known for its excellent video capabilities and robust L-mount system (shared with Leica and Sigma). The Lumix S series offers full-frame mirrorless cameras with strong hybrid photo/video performance.

## Key Series
- **Lumix S Series**: Full-frame L-mount mirrorless (S1, S1R, S5, S5 II, S5 IIX)
- **Lumix GH Series**: Micro Four Thirds with exceptional video features (GH6, GH5 II)
- **Lumix G Series**: Micro Four Thirds stills-focused cameras

## Common Menu Structure
Panasonic uses a logical menu system organized by function:
- **Photo Style**: Picture profiles and color settings
- **Focus**: AF modes, focus areas, custom focus settings  
- **Exposure**: ISO, shutter speed, aperture control
- **Motion Picture**: Video-specific settings (V-Log, frame rates, codecs)
- **Custom**: User-defined settings and shortcuts

## Key Features
- **Dual Native ISO**: Excellent low-light performance with two base ISO values
- **V-Log/V-Gamut**: Professional video color profiles (available on higher-end models)
- **In-Body Image Stabilization**: 5-axis IBIS on most models
- **Anamorphic Support**: Specialized modes for cinematic widescreen video
- **High Res Mode**: Pixel-shift for ultra-high resolution stills (S1R)

## Common Operations
### Accessing Photo Styles
Menu → Photo Style → Select desired profile (Standard, Vivid, Natural, etc.)

### Setting Dual Native ISO
Auto ISO will automatically use the appropriate native ISO based on lighting conditions

### Enabling V-Log
Requires paid upgrade on some models; Menu → Motion Picture → Picture Profile → V-Log

### Customizing Function Buttons
Menu → Custom → Fn Button Settings → Assign desired functions

## Popular Models for Learning
- **Lumix S5 II**: Best entry-level full-frame with excellent autofocus
- **Lumix GH6**: Ultimate video-centric camera with professional features
- **Lumix S1**: Flagship stills camera with high resolution and build quality

## Learning Tips
- Start with iA (Intelligent Auto) mode to understand basic operations
- Explore Photo Styles before diving into manual exposure
- Use the Q.Menu button for quick access to frequently changed settings
- Practice with 4K Photo modes for action photography
