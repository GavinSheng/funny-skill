# Hasselblad H6D-100c

## 基本规格
- **传感器**: 100MP 中画幅 CMOS (53.4 x 40.0mm)
- **ISO范围**: 64-12800
- **连拍速度**: 1.5 fps
- **视频**: 无视频功能
- **取景器**: 光学腰平取景器 + 电子取景器选项
- **存储**: CFast 2.0 + SD 卡

## 核心特点
- **中画幅专业级**: 专为商业摄影和高端工作室设计
- **True Focus II**: 自动对焦系统，确保精确对焦
- **超高质量**: 16-bit 色彩深度，15档动态范围
- **模块化设计**: 可更换镜头、取景器、数码后背

## 操作要点
- **工作流程**: 主要用于静态拍摄，不适合快速移动场景
- **对焦方式**: 手动对焦为主，配合 True Focus II 系统
- **曝光控制**: 手动模式为主，支持 TTL 测光
- **连接性**: USB 3.0, FireWire, 支持 Phocus 软件

## 适用场景
- 商业产品摄影
- 高端人像摄影  
- 艺术摄影
- 建筑摄影
- 风光摄影（三脚架）

## 注意事项
- 价格昂贵，主要面向专业用户
- 体积较大，便携性较差
- 电池续航相对较短
- 需要专用软件处理 RAW 文件
