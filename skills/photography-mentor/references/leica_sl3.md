# Leica SL3 差异化特性

## 核心规格与定位
- **传感器**: 60MP 全画幅 BSI CMOS
- **处理器**: Maestro IV，提供更快的处理速度和更好的AI功能
- **视频能力**: 8K60p 10-bit 4:2:2 内录，支持ProRes RAW外录
- **对焦系统**: 改进的混合AF系统，包含相位检测点
- **机身设计**: 经典Leica设计语言，镁合金机身，IP54防护等级

## 菜单系统特色
- **触摸优先**: 6.0英寸高分辨率触摸屏，支持完整的触摸操作
- **直观布局**: 简化的菜单结构，常用设置快速访问
- **自定义界面**: 可创建个人化的控制面板

## 操作要点
### 开机与基本设置
1. 电源开关位于快门按钮周围
2. 首次使用建议通过菜单进行固件更新
3. L-mount镜头自动识别，无需手动设置

### 拍摄模式选择
- **PASM模式**: 通过模式转盘选择
- **场景模式**: 包括人像、风景、运动等预设
- **自定义模式**: C1-C3可保存个人偏好设置

### 对焦操作
- **AF-S/AF-C切换**: 菜单中设置或通过Fn按钮
- **对焦点选择**: 触摸屏幕直接选择或使用方向键
- **眼部对焦**: 支持人眼、动物眼自动识别

### 视频拍摄
- **Log格式**: 支持Leica L-Log，提供14+档动态范围
- **音频控制**: 内置立体声麦克风，支持外接XLR适配器
- **散热管理**: 主动散热系统支持长时间8K录制

## 与其他Leica相机的区别
- **相比SL2**: 更高分辨率、更好的视频能力、改进的AF系统
- **相比Q3**: 可换镜头vs固定镜头，更专业的操控
- **相比M11**: 电子取景vs光学取景，自动对焦vs手动对焦

## 常见问题解决
- **电池续航**: 使用LP-SCL7电池，建议携带备用电池
- **存储卡**: 支持CFexpress Type B和SD UHS-II双卡槽
- **固件更新**: 通过Leica Fotos app或官网下载最新固件
