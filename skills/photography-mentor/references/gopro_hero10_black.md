# GoPro HERO10 Black 相机指南

## 基本规格
- **传感器**: 23MP GP2 处理器
- **视频**: 5.3K60, 4K120, 2.7K240
- **照片**: 23MP 静态照片
- **防抖**: HyperSmooth 4.0
- **防水**: 33英尺（10米）无需外壳

## 核心功能特点

### GP2 处理器优势
- 更快的启动和处理速度
- 改进的低光性能
- 更高效的电池管理
- 支持更高帧率录制

### 视频模式
- **5.3K60**: 最高分辨率，适合后期裁剪
- **4K120**: 高帧率慢动作，流畅度极佳  
- **2.7K240**: 超级慢动作，适合动作捕捉
- **1080p240**: 极限慢动作效果

### HyperSmooth 4.0 防抖
- 所有视频模式都支持
- Horizon Leveling 地平线校正
- Boost 模式提供更强稳定效果
- 数字镜头畸变校正

## 实用操作技巧

### 电池优化
- 使用 Enduro 电池提升低温性能
- 关闭 GPS 和 Wi-Fi 节省电量
- 降低分辨率和帧率延长录制时间
- 启用自动关机功能

### 拍摄设置建议
- **运动场景**: 4K60 + HyperSmooth Boost
- **慢动作**: 2.7K120 或 1080p240  
- **低光环境**: 4K30 + Max Lens Mod
- **Vlog**: Linear + Horizon Lock

## 常见问题解决

### 过热问题
- 避免长时间 5.3K 录制
- 在阴凉处使用
- 移除保护壳改善散热
- 降低视频设置

### 存储卡选择
- 推荐 V30 或更高规格 microSD
- UHS-I Speed Class 3 (U3)
- 容量建议 128GB+ 用于 5.3K 录制
- 定期格式化存储卡

## 配件生态系统
- **Max Lens Mod**: 超宽视角 + 数字镜头
- **Media Mod**: 麦克风 + HDMI + 冷靴  
- **Light Mod**: LED 补光灯
- **Volta**: 电池手柄 + 三脚架
- **各种安装支架**: 胸带、头带、吸盘等
