# 人像摄影场景指南

## 通用设置原则

### 光圈选择
- **全画幅相机**: f/1.2 - f/2.8 (浅景深)
- **APS-C相机**: f/1.4 - f/2.0 (等效全画幅f/2.0 - f/3.2)
- **M4/3相机**: f/1.2 - f/1.8 (等效全画幅f/2.4 - f/3.6)

### 快门速度
- **静态人像**: 1/125s 或更快
- **动态人像**: 1/250s 或更快
- **带防抖**: 可适当降低快门速度

### ISO设置
- **充足光线**: ISO 100-400
- **室内/弱光**: ISO 800-3200
- **夜景**: ISO 3200+

## 各品牌具体操作

### Canon 相机
**EOS R5/R5 Mark II/R6 Mark III:**
- 模式转盘：Av (光圈优先)
- 设置光圈：前拨轮
- 眼部AF：MENU → AF → 人脸/眼部检测 → 开启
- 背景虚化预览：按景深预览按钮

**EOS 5D Mark III:**
- 模式转盘：Av
- 设置光圈：主拨轮
- AI Servo + 单点AF 用于移动主体

### Sony 相机
**Alpha A7 IV/A7R V/A7C II:**
- 模式转盘：A (光圈优先)
- 设置光圈：前/后拨轮 (取决于设置)
- 实时眼部AF：MENU → AF → 人脸/眼部优先 → 开启
- 创意外观：可直接应用胶片模拟效果

**Alpha A9 III:**
- 高速连拍模式下仍保持眼部追踪
- 电子快门无果冻效应

### Nikon 相机
**Z8/Z9/Zf:**
- 模式转盘：A (光圈优先)
- 设置光圈：副指令拨盘
- 3D跟踪AF：自动识别并跟踪主体
- 人像优化校正：减少面部变形

### Fujifilm 相机
**X-T5/X-H2/X-H2S:**
- 模式转盘：A (光圈优先)
- 光圈环：镜头上的物理光圈环
- 胶片模拟：Classic Neg, Provia, Astia 等适合人像
- 眼部AF：AF模式 → 人脸/眼部检测

### Panasonic 相机
**S5 II/S5 IIX:**
- 模式转盘：A (光圈优先)
- 设置光圈：前拨轮
- DFD对焦：快速眼部识别
- LUT应用：可加载自定义色彩配置文件

### OM System 相机
**OM-1 Mark II/OM-5:**
- 模式转盘：A (光圈优先)
- 设置光圈：前拨轮
- 人脸/眼部优先AF：高速精准
- Live ND：可模拟慢门效果

## 常见问题解决

### 背景不够虚化
- **全画幅**: 使用更大光圈或更长焦距
- **APS-C**: 靠近主体，使用50mm以上焦距
- **M4/3**: 使用f/1.2-f/1.8镜头，靠近主体

### 眼部对焦不工作
- 确认AF模式设置为人脸/眼部检测
- 主体不要太小（占画面1/4以上）
- 光线充足，避免逆光

### 肤色偏色
- 使用自定义白平衡
- Fujifilm：选择Provia或Classic Chrome
- Sony：调整肤色色调设置
- Canon：使用人像图片样式
