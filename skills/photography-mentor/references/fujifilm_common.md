# Fujifilm Brand Common Operations

## Overview
Fujifilm is renowned for its film simulation modes, excellent color science, and retro-inspired design. The X-series (APS-C) and GFX-series (medium format) are their main camera lines.

## Key Features
- **Film Simulation**: Unique color profiles mimicking classic Fujifilm films
- **Retro Design**: Physical dials for shutter speed, ISO, and exposure compensation
- **Color Science**: Excellent JPEG output with minimal post-processing needed
- **Hybrid AF**: Combination of phase detection and contrast detection

## Menu System
- **Q Menu**: Quick access to frequently used settings
- **Main Menu**: Organized into Shooting, Playback, and Setup tabs
- **My Menu**: Customizable menu for personal preferences

## Common Operations
- **Exposure Compensation**: Dedicated dial on most models
- **ISO Settings**: Physical dial or menu setting
- **White Balance**: Accessible via Q menu or main menu
- **Focus Modes**: Single Point, Zone, Wide/Tracking

## Popular Models (2025-2026)
- **X-T5**: Flagship APS-C with 40MP sensor
- **X-H2**: High-resolution APS-C with 40MP sensor
- **X-S20**: Mid-range APS-C with excellent video capabilities
- **GFX100 II**: Medium format with 102MP sensor
- **GFX50S II**: Affordable medium format option
