# Ricoh 相机通用操作指南

## 品牌特点
- **街拍专家**: GR 系列被誉为"街拍神器"
- **便携设计**: 紧凑机身，APS-C 传感器
- **操控简洁**: 物理拨盘 + 快捷键
- **胶片模拟**: 内置多种胶片效果

## 菜单系统
- **主菜单**: 按 MENU 键进入
- **快速设置**: ADJ./OK 按钮
- **Fn 按钮**: 可自定义功能
- **前后拨盘**: 参数调整

## 核心功能
- **Snap Focus**: 预设对焦距离（2m/2.5m/5m）
- **ND 滤镜**: 内置中性密度滤镜
- **Hybrid AF**: 相位+对比度混合对焦
- **AA Filter Simulator**: 抗锯齿滤镜模拟器

## 操作逻辑
- **曝光补偿**: 前拨盘直接调整
- **ISO 设置**: 后拨盘 + Fn 按钮
- **白平衡**: MENU → 拍摄设置 → 白平衡
- **胶片效果**: MENU → 图像设置 → 图像控制

## 特殊功能
- **间隔拍摄**: 内置延时摄影
- **多重曝光**: 可叠加多张照片
- **水平仪**: 电子水平显示
- **Wi-Fi 连接**: 手机遥控和传输
