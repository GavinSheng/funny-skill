# OM System OM-1 Mark II 相机指南

## 基本规格
- **传感器**: 20.4MP Live MOS Micro Four Thirds
- **图像处理器**: TruePic X
- **防抖系统**: 7.5档 5轴机身防抖
- **连拍速度**: 50fps (电子快门), 10fps (机械快门)
- **视频能力**: 4K 60p, C4K 30p, 10-bit 4:2:2
- **对焦系统**: AI主体检测对焦 (人/动物/车辆)

## 核心操作特点

### 菜单系统
OM System 使用直观的菜单布局，主要分为：
- **拍摄菜单**: 图像质量、白平衡、ISO等
- **对焦菜单**: 对焦模式、对焦点选择、AI设置
- **视频菜单**: 视频格式、帧率、编码
- **自定义菜单**: 按钮分配、快捷设置

### 快速设置方法
1. **ISO设置**: 按下ISO按钮，转前后拨盘
2. **白平衡**: WB按钮 + 前拨盘选择预设，后拨盘微调
3. **对焦模式**: AF/MF切换杆 + OK按钮进入详细设置
4. **驱动模式**: DRIVE按钮选择单张/连拍/自拍

## 特色功能

### AI主体检测
- **人眼/人脸检测**: 自动识别人眼和面部
- **动物检测**: 支持鸟类、哺乳动物等
- **车辆检测**: 汽车、摩托车、飞机识别
- **激活方法**: MENU → AF/MF → AF Target → Subject Detection

### 高级防抖
- **同步IS**: 与兼容镜头协同工作
- **手持高分辨率**: 80MP合成模式
- **防抖关闭时机**: 使用三脚架或高速快门时

### 视频功能
- **ProRes RAW**: 通过HDMI输出到外录设备
- **Log曲线**: OM-Log400提供14+档动态范围
- **视频裁切**: Super 35mm模式可用

## 常见问题解决

### 电池续航优化
- 关闭Wi-Fi/蓝牙
- 降低LCD亮度
- 使用省电模式
- 备用电池推荐：BLX-1

### 存储卡建议
- **照片**: UHS-II V60以上
- **4K视频**: UHS-II V90
- **双卡槽**: 可设置备份或溢出

### 镜头兼容性
- **原生M.Zuiko**: 最佳性能
- **老款Four Thirds**: 通过MMF-3适配器
- **第三方镜头**: 部分支持，功能可能受限

## 实战拍摄建议

### 风光摄影
- 使用三脚架 + 低ISO (200)
- 开启高分辨率模式 (80MP)
- 使用ND滤镜控制曝光

### 野生动物
- AI动物检测 + 连拍50fps
- 预拍摄功能 (最多35张)
- 防水防尘机身适合户外

### 视频拍摄
- OM-Log400 + 外录ProRes RAW
- 使用IBIS稳定画面
- 监控耳机监听音频
