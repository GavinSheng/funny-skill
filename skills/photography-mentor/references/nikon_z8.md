# Nikon Z8 差异化特性

## 与Nikon通用操作的主要差异

### 旗舰级性能
- **45.7MP传感器**: 全画幅背照式CMOS
- **EXPEED 7处理器**: 双处理器架构
- **8K视频**: 8K 30p 12-bit RAW内录
- **ProRes支持**: 内录ProRes 422 HQ，外录ProRes RAW

### 对焦系统升级
- **3D追踪增强**: 改进的AI主体识别算法
- **眼部检测**: 更精准的人眼/动物眼检测
- **车辆识别**: 汽车、摩托车、飞机、火车专用识别
- **低光对焦**: -9EV扩展对焦范围（业界领先）

### 视频能力突破
- **无裁切8K**: 真正的无裁切8K录制
- **N-Log优化**: 改进的动态范围表现
- **散热系统**: 专业级散热设计延长录制时间
- **音频输入**: 内置高品质麦克风和耳机监听

### 专业工作流
- **双卡槽**: CFexpress Type B + SD UHS-II
- **USB供电**: 支持USB-C PD供电和充电
- **FTP传输**: 内置无线FTP文件传输
- **PC遥控**: 增强的电脑远程控制功能

### 操作体验
- **高刷新率EVF**: 369万点OLED EVF，120fps刷新率
- **侧翻屏**: 适合视频拍摄的侧翻触摸屏
- **防尘防滴**: 专业级防护设计
- **电池续航**: 改进的电池管理系统

## 操作要点
- **8K录制**: 需要CFexpress Type B卡，注意散热管理
- **ProRes内录**: 在视频菜单中选择ProRes 422 HQ选项
- **3D追踪**: 在自定义设定菜单中启用AI主体识别
- **双卡槽**: 建议CFexpress用于RAW+视频，SD用于JPEG备份
