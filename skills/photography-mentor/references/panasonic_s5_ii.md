# Panasonic Lumix S5 II 相机指南

## 基本规格
- **传感器**: 24.2MP 全画幅 CMOS
- **处理器**: L2 处理器
- **视频**: 6K 30p, 4K 60p, C4K 30p
- **防抖**: 5轴机身防抖 (7.5档补偿)
- **相位对焦**: 首次在L卡口全画幅相机上加入相位检测AF
- **价格区间**: 约 $2000 (机身)

## 核心特色功能

### 混合相位/对比度对焦系统
- **相位检测点**: 覆盖传感器90%区域
- **人眼/动物眼检测**: 支持照片和视频模式
- **主体识别**: 汽车、飞机、鸟类等
- **低光对焦**: -6EV 极限对焦能力

### 视频功能
- **ProRes录制**: 内录 ProRes HQ (需CFexpress B卡)
- **V-Log/V-Gamut**: 14+档动态范围
- **Anamorphic模式**: 支持变形镜头拍摄
- **实时LUT**: 可加载自定义LUT进行监看

### 拍摄模式
- **6K Photo**: 30fps 连拍，可提取18MP静态图像
- **Post Focus**: 后期选择焦点
- **Focus Stacking**: 自动景深合成

## 常用操作指南

### 快速设置人眼对焦
1. 按 `MENU` 进入菜单
2. 选择 `AF/AE` → `AF模式` → `人脸/眼检测AF`
3. 设置 `检测对象` → `人`
4. 在 `自定义设置` 中分配到Fn按钮

### V-Log设置
1. `MENU` → `Motion Picture` → `Photo Style`
2. 选择 `V-Log`
3. 调整 `Master Pedestal` 和 `Gamma` 设置
4. 建议使用外录监视器查看LUT效果

### 防抖优化
- 搭配OIS镜头时：启用 `Dual I.S.2`
- 手持拍摄：使用 `Active` 防抖模式
- 三脚架拍摄：关闭机身防抖

## 适用场景推荐

### 视频创作者
- **优势**: 优秀的视频规格、相位对焦、防抖
- **推荐设置**: V-Log + ProRes + 人眼对焦
- **配件建议**: CFexpress B卡、外录监视器

### 摄影爱好者
- **优势**: 全画幅画质、防抖、6K Photo
- **推荐设置**: Standard色彩模式 + 连拍模式
- **镜头建议**: Lumix S 24-105mm f/4

### 混合用户
- **优势**: 照片视频兼顾、坚固机身、电池续航
- **推荐设置**: 根据拍摄需求快速切换模式
- **工作流**: 利用Wi-Fi传输和远程控制

## 常见问题解决

### 对焦不准确
- 检查AF模式是否正确设置
- 确保光线充足（低于-6EV可能失效）
- 尝试调整AF灵敏度设置

### 电池续航短
- 关闭Wi-Fi和蓝牙
- 降低LCD亮度
- 使用节能模式
- 购买额外电池或手柄

### 视频过热
- 避免长时间录制6K
- 在凉爽环境中拍摄
- 使用散热配件
- 间隔录制避免连续工作
