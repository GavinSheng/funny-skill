# Canon EOS R1 (2025) 专属操作指南

## 核心特性
- **24MP 全画幅传感器**：专为高速连拍和视频优化
- **8K120p 视频录制**：支持ProRes RAW外录
- **高级AI主体识别**：可识别人、动物、车辆等多种主体
- **专业级机身**：镁合金机身，防尘防滴，双CFexpress B卡槽

## 关键操作步骤

### 1. AI主体检测设置
```
菜单 → 拍摄设置 → 主体检测 → [选择主体类型]
- 人物：面部+眼部+头部+身体追踪
- 动物：猫/狗/鸟类的眼部追踪  
- 车辆：赛车/飞机/火车的特定部位追踪
```

### 2. 高速连拍设置
```
驱动模式 → 高速连拍+ → 最高30fps（电子快门）
- 使用机械快门：最高12fps
- RAW连拍缓冲：约240张（CFexpress）
```

### 3. 8K视频设置
```
菜单 → 视频拍摄 → 视频记录尺寸 → 8K DCI (8192×4320)
- 帧率：23.98/25/29.97fps
- 码率：最高800Mbps（ALL-I）
- 色彩采样：4:2:2 10-bit
```

### 4. 专业工作流程
- **双卡槽备份**：支持独立/备份/分类存储
- **USB-C供电**：支持PD充电和供电拍摄
- **无线传输**：内置Wi-Fi 6和蓝牙5.3

## 与R5 Mark II的主要区别
- **更高连拍速度**：30fps vs 20fps
- **更好的视频规格**：8K120p vs 8K60p
- **专业级机身**：更坚固的构造和操控
- **双CFexpress**：vs 单CFexpress + SD
- **价格定位**：旗舰专业级 vs 高端专业级
