# DJI Osmo Action 2

## 基本规格
- **传感器**: 1/1.7英寸 CMOS
- **分辨率**: 4K/60fps, 2.7K/120fps, 1080p/240fps
- **镜头**: 155° 超广角 FOV
- **防抖**: RockSteady 2.0 + HorizonSteady
- **防水**: 16米裸机防水
- **电池**: 可更换电池，约160分钟续航
- **特色功能**: 磁吸快拆设计、双屏显示、AI剪辑

## 核心操作指南

### 开机/关机
- **开机**: 长按电源键2秒
- **关机**: 在主界面长按电源键2秒
- **快速启动**: 按下任意按钮即可从待机唤醒

### 拍摄模式切换
- **视频模式**: 默认模式，支持多种分辨率和帧率
- **照片模式**: 单拍、连拍、定时拍摄
- **延时摄影**: 移动延时、定点延时、轨迹延时
- **慢动作**: 最高240fps慢动作录制

### 关键设置
- **分辨率/帧率**: 设置 → 视频 → 分辨率/帧率
- **防抖设置**: 设置 → 防抖 → RockSteady/HorizonSteady/关闭
- **色彩模式**: D-Cinelike（专业后期）、普通（直出）
- **曝光补偿**: 在拍摄界面滑动右侧EV滑块

### 磁吸快拆系统
- **安装**: 将相机靠近磁吸配件自动吸附
- **拆卸**: 轻轻旋转即可分离
- **兼容性**: 支持所有Osmo Action磁吸配件

### 双屏使用
- **前屏**: 1.4英寸触摸屏，用于自拍构图
- **后屏**: 2.25英寸触摸屏，主操作界面
- **屏幕切换**: 双击前屏或在设置中启用

### AI剪辑功能
- **自动剪辑**: 录制完成后自动分析精彩片段
- **模板选择**: 多种预设模板可选
- **音乐配乐**: 内置版权音乐库
- **导出分享**: 直接导出到手机相册或社交平台

## 实用拍摄技巧

### 运动场景
- 使用HorizonSteady保持地平线水平
- 选择4K/60fps获得最佳画质和慢动作素材
- 启用D-Cinelike模式为后期保留更多动态范围

### 水下拍摄
- 裸机防水16米，无需额外防水壳
- 水下建议使用1080p/60fps减少色散
- 注意清洁镜头表面避免水渍影响画质

### 自拍Vlog
- 利用前屏进行构图
- 使用磁吸配件快速安装到自拍杆
- 开启人脸追踪确保主体清晰

## 常见问题解决

### 电池续航短
- 降低分辨率和帧率设置
- 关闭不必要的功能（如GPS、WiFi）
- 准备备用电池

### 画面模糊
- 检查镜头是否清洁
- 确保防抖功能已开启
- 避免在低光环境下使用过高ISO

### 连接手机失败
- 确保DJI Mimo App已更新到最新版本
- 重启相机和手机蓝牙/WiFi
- 重置相机网络设置
