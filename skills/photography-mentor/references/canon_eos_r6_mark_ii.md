# Canon EOS R6 Mark II

## 基本规格
- **传感器**: 24.2MP 全画幅 CMOS
- **处理器**: DIGIC X
- **连拍速度**: 12 fps (机械快门), 40 fps (电子快门)
- **视频**: 6K 超采样 4K 60p, 10-bit 4:2:2
- **防抖**: 5轴机身防抖，最高8档补偿

## 核心功能特点

### 自动对焦系统
- **Dual Pixel CMOS AF II**: 覆盖约100%画面
- **人眼/动物眼/车辆识别**: 支持汽车、摩托车检测
- **主体识别追踪**: 智能主体识别和持续追踪

### 视频能力
- **无裁切4K 60p**: Full HD 180p 慢动作
- **Canon Log 3**: 提供12+档动态范围
- **HDR PQ**: 内置HDR视频录制

### 操作界面
- **菜单系统**: 与R5/R3统一的菜单逻辑
- **自定义按钮**: 多个可自定义功能按钮
- **触摸屏**: 可翻转LCD触摸屏

## 常用设置路径

### 设置眼部对焦
1. MENU → AF → Subject to detect → Eye (People/Animals)
2. 或使用快捷键：按住AF-ON按钮 + 转动主拨盘

### 视频录制设置
1. MENU → Movie → Movie recording size → 4K 60p
2. MENU → Movie → Canon Log settings → Canon Log 3

### 防抖设置
1. MENU → IS → IS mode → Dynamic IS (视频) / Standard IS (照片)

## 适用场景
- **婚礼摄影**: 快速连拍和可靠对焦
- **野生动物**: 动物眼识别和高速连拍
- **视频创作**: 专业级视频功能和Log格式
- **体育摄影**: 高速连拍和主体追踪
