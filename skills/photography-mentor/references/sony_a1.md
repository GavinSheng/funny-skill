# Sony Alpha A1

## 基本规格
- **传感器**: 5010万像素全画幅堆栈式Exmor RS CMOS
- **处理器**: BIONZ XR双处理器
- **连拍速度**: 30张/秒（电子快门），带AF/AE追踪
- **视频**: 8K 30p 10-bit 4:2:2, 4K 120p
- **防抖**: 5轴机身防抖，最高5.5档补偿

## 核心功能特点

### 高速性能
- 全球最快的30fps连拍，无黑屏
- 120次/秒AF计算，759点相位检测对焦
- 实时眼部对焦（人眼、动物眼、鸟眼）

### 视频能力
- 8K 30p超采样录制（约8.6K超采样）
- 4K 120p慢动作，支持S-Log3和HLG
- 专业音频输入：XLR适配器支持

### 操作界面
- 双卡槽：CFexpress Type A + SD UHS-II
- 高分辨率EVF：944万点OLED，240fps刷新率
- 耐用性：镁合金机身，防尘防滴设计

## 品牌通用操作继承
- 菜单系统：Sony标准菜单布局
- 快捷菜单：Fn按钮自定义设置
- 对焦模式：AF-S/AF-C/MF切换逻辑
- 曝光模式：P/A/S/M/B模式操作

## 实用拍摄技巧

### 体育摄影设置
- AF模式：AF-C + 广域对焦
- 连拍：Hi+模式，30fps
- 曝光：快门优先，1/1000s以上

### 视频拍摄设置  
- 分辨率：8K 30p或4K 60p
- 色彩：S-Log3 + S-Gamut3.Cine
- 音频：外接麦克风，手动电平控制

## 常见问题解决

### 过热问题
- 8K录制建议使用散热配件
- 环境温度超过30°C时注意录制时间限制
- 启用自动关机保护功能

### 存储卡选择
- CFexpress Type A卡推荐：Sony TOUGH系列
- SD卡要求：V90等级，UHS-II
- 双卡备份：建议不同品牌避免同时故障
