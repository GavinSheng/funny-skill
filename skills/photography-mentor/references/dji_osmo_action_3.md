# DJI Osmo Action 3 运动相机指南

## 核心规格
- **传感器**: 1/1.7英寸 CMOS
- **分辨率**: 4K/120fps, 2.7K/120fps, 1080p/240fps
- **防抖**: RockSteady 3.0 + HorizonSteady 地平线增稳
- **防水**: 16米裸机防水（无需防水壳）
- **电池**: 1770mAh，支持快充，低温续航优化
- **特色功能**: 竖拍模式、磁吸快拆设计、双屏显示

## 基础操作设置

### 开机/关机
- **开机**: 长按电源键2秒
- **关机**: 在关机状态下长按电源键2秒
- **快速开机**: 关机状态下按快门键可快速启动并开始录像

### 拍摄模式切换
- **照片/视频切换**: 上下滑动主屏幕或按模式按钮
- **慢动作**: 在视频模式下选择1080p/240fps
- **延时摄影**: 进入专业模式选择延时选项

### 双屏使用
- **前屏**: 1.4英寸触摸屏，用于自拍构图
- **后屏**: 2.25英寸触摸屏，主操作界面
- **屏幕锁定**: 双击电源键锁定触摸屏防止误触

## 高级功能设置

### HorizonSteady 地平线增稳
- **开启方法**: 设置 → 视频 → 地平线增稳 → 开启
- **效果**: 即使相机旋转360度，画面始终保持水平
- **限制**: 仅支持2.7K及以下分辨率

### 专业模式参数调整
- **ISO范围**: 100-3200
- **快门速度**: 1/8000s - 30s
- **白平衡**: 自动、晴天、阴天、白炽灯、荧光灯、自定义
- **色彩模式**: DJI色彩、D-Cinelike

### 磁吸配件系统
- **快拆设计**: 支持磁吸式配件快速安装
- **兼容配件**: 胸带、头带、自拍杆、浮力手柄等
- **扩展坞**: 支持外接麦克风和额外电池

## 实用拍摄技巧

### 极限运动拍摄
- **固定位置**: 使用各种支架确保相机稳定
- **广角镜头**: 充分利用超广角视野捕捉动态
- **高帧率**: 使用120fps以上帧率获得流畅慢动作

### 水下拍摄
- **裸机防水**: 最深16米，无需额外防水壳
- **色彩校正**: 水下建议使用D-Cinelike模式便于后期调色
- **防雾处理**: 在潮湿环境中注意镜头起雾问题

### 夜间拍摄
- **低光性能**: 1/1.7英寸传感器提供较好夜景表现
- **防抖效果**: RockSteady在低光下依然有效
- **补光建议**: 可外接小型LED补光灯

## 电池与存储管理

### 电池续航
- **4K/60fps**: 约150分钟
- **1080p/30fps**: 约160分钟  
- **低温环境**: 专用电池仓设计，-20°C仍可正常工作

### 存储卡要求
- **推荐卡型**: UHS-I Speed Class 3 (U3) 或更高
- **最大容量**: 支持microSDXC，最高1TB
- **格式化**: 建议在相机内格式化存储卡

## 常见问题解决

### 相机过热
- **原因**: 长时间4K录制导致
- **解决**: 降低分辨率或帧率，避免阳光直射

### 触摸屏失灵
- **原因**: 水滴或手套影响
- **解决**: 使用物理按键，或擦干屏幕后操作

### 连接手机失败
- **步骤**: 确保DJI Mimo App最新版本，重启蓝牙和WiFi
