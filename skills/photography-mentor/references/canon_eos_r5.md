# Canon EOS R5 差异化特性

## 与Canon通用操作的主要差异

### 视频能力突破
- **8K RAW**: 30fps, 12-bit Cinema RAW Light内录
- **4K HQ**: 超采样8K，提供极致4K画质
- **ProRes支持**: 外录Apple ProRes RAW/HQ
- **散热管理**: 连续录制时间限制，建议准备备用相机

### 对焦系统升级
- **双像素CMOS AF II**: 1053个自动对焦点，覆盖约100%画面
- **人眼/动物眼检测**: 支持鸟类眼部追踪
- **车辆识别**: 汽车、摩托车自动识别追踪
- **低光对焦**: -6EV扩展对焦范围

### 机身防抖 (IBIS)
- **最高8级防抖**: 与RF镜头协同工作
- **组合防抖**: 机身+镜头防抖协同
- **视频防抖**: 数字防抖可选（会裁剪画面）

### 连拍性能
- **电子快门**: 20fps连拍，无黑屏，静音拍摄
- **机械快门**: 12fps连拍
- **缓冲深度**: 显著增加的RAW连拍张数

### 操作体验
- **电子取景器**: 576万点OLED EVF，120fps刷新率
- **翻转触摸屏**: 适合视频拍摄的侧翻屏
- **多功能条**: 可自定义滑动/点击功能
- **双卡槽**: CFexpress Type B + SD UHS-II

### 电池和存储
- **LP-E6NH电池**: 新型高容量电池
- **USB供电**: 支持USB-C PD充电和供电
- **CFexpress Type B**: 推荐用于8K和高速连拍
- **SD UHS-II**: 适用于4K及以下规格

## 操作要点
- **8K录制**: 注意散热限制，建议间歇录制
- **CFexpress卡**: 8K和高速连拍必需，选择V90等级
- **IBIS设置**: 在菜单中可调整防抖强度和模式
- **双像素AF**: 在视频模式下可使用面部+追踪
- **电子快门**: 避免在荧光灯下使用，可能产生条纹
