# Hasselblad X2D 100C

## 基本规格
- **传感器**: 1亿像素中画幅 BSI CMOS (43.8 x 32.9mm)
- **处理器**: Hasselblad 自研图像处理器
- **ISO范围**: 64-12800 (原生)
- **连拍速度**: 3.7 fps
- **视频**: 无视频功能（纯静态相机）

## 核心特色功能

### 中画幅优势
- 超高分辨率：1亿像素提供极致细节
- 优秀动态范围：14+档动态范围
- 自然色彩科学：Hasselblad Natural Colour Solution (HNCS)

### 独特设计
- 内置1TB SSD存储
- 五轴机身防抖 (IBIS)
- 高分辨率电子取景器 (EVF)
- 触摸屏操作界面

## 操作要点

### 拍摄设置
- 使用Phocus Mobile 2进行无线控制
- 支持CFexpress Type B和SD双卡槽
- 电池续航约400张照片

### 工作流程
- 原生支持16-bit RAW格式
- 与Adobe Lightroom和Capture One完美兼容
- Hasselblad Phocus桌面软件提供完整后期支持

## 适用场景
- 商业摄影（产品、时尚、人像）
- 风光摄影（需要极致细节）
- 艺术摄影（追求最高画质）

## 注意事项
- 体积相对较大，不适合快速抓拍
- 价格昂贵，主要面向专业用户
- 镜头系统相对有限但素质极高
