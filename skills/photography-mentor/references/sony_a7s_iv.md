# Sony Alpha A7S IV (2025) 差异化特性

## 核心定位
A7S IV是索尼专业视频系列的最新旗舰，专为低光环境和高质量视频拍摄设计。

## 关键操作差异

### 视频设置
- **8K视频录制**: 支持8K 30p 10-bit 4:2:2内录
- **S-Log3/S-Gamut3.Cine**: 提供15+档动态范围
- **ProRes RAW输出**: 通过HDMI输出到外录设备
- **散热系统**: 改进的主动散热，支持长时间8K录制

### 低光性能
- **12MP全画幅传感器**: 优化像素尺寸获得最佳低光表现
- **ISO扩展**: 原生ISO 80-102400，可扩展至409600
- **AI降噪**: 新的AI处理引擎提供更好的高ISO降噪

### 对焦系统
- **实时眼部AF**: 支持人眼、动物眼、鸟类眼追踪
- **视频AF优化**: 专门针对视频拍摄优化的对焦算法
- **触摸跟踪**: 支持触摸屏选择和跟踪主体

### 专业功能
- **时间码同步**: 内置时间码发生器，支持多机同步
- **双原生ISO**: ISO 800和ISO 12800提供最佳信噪比
- **LUT加载**: 直接在相机内加载自定义LUT

## 快速设置指南

### 电影工作流设置
1. 进入菜单 → 拍摄设置 → 文件格式 → 选择XAVC HS或ProRes
2. 菜单 → 曝光/颜色 → Gamma显示辅助 → 开启S-Log3
3. 菜单 → 对焦 → AF-C优先级 → 设置为"对焦"
4. 菜单 → 网络 → FTP传输 → 配置自动备份

### 低光拍摄设置  
1. ISO设置为800或12800（双原生ISO点）
2. 使用S-Log3伽马曲线获得最大动态范围
3. 启用AI降噪功能（菜单 → 画质 → 降噪）
4. 使用实时眼部AF确保精准对焦

## 注意事项
- A7S IV主要面向视频创作者，静态照片功能相对简化
- 8K录制时注意存储卡速度要求（CFexpress Type A）
- 散热系统在高温环境下可能限制录制时间
- 建议搭配外录设备获得最佳视频质量
