# 有趣的skill

闲暇时搞的几个有趣的 skill，将专项能力封装为模块化 skill，可配合 AI Agent 使用。

## 安装与使用

将 `skills/` 目录放置到 Agent 可识别的 skill 路径下即可自动加载。每个 skill 包含独立的 `SKILL.md` 定义、脚本和参考资料。

---

## 🍳 cooking-recipes | 烹饪菜谱助手

智能菜谱助手，支持中英双语，覆盖中国八大菜系及多国料理。

**支持菜系**：川菜、粤菜、鲁菜、苏菜、浙菜、闽菜、湘菜、徽菜，以及泰国菜、法国菜、意大利菜、日本料理、韩国料理、印度菜、墨西哥菜、地中海菜等。

**功能**：
- 按菜系搜索和浏览菜谱
- 提供详细的食材清单、步骤和烹饪技巧
- 标注难度等级、准备时间和烹饪时间
- 支持本地菜谱数据库检索和联网搜索

### 使用场景与示例

<details>
<summary><b>🍽️ 学做一道菜</b> — 告诉我想做什么，我给你步骤</summary>

| 输入 | 输出 |
|------|------|
| `教我做宫保鸡丁` | 完整菜谱：食材用量 + 分步做法 + 技巧提示 |
| `红烧肉怎么做？` | 家常红烧肉的详细步骤和注意事项 |
| `麻婆豆腐的做法` | 正宗川菜麻婆豆腐的制作流程 |
</details>

<details>
<summary><b>🌍 菜谱推荐</b> — 没有目标菜，按菜系/口味/场景推荐</summary>

| 输入 | 输出 |
|------|------|
| `有什么川菜推荐？` | 多道经典川菜列表，附难度和时间 |
| `推荐一个泰国菜` | 泰国菜推荐（如冬阴功汤、泰式炒河粉等） |
| `有没有适合夏天吃的凉菜？` | 清爽凉菜推荐 |
| `推荐几道适合请客的大菜` | 拿得出手的宴客菜推荐 |
</details>

<details>
<summary><b>🥘 食材与技巧</b> — 查食材搭配、替代方案、烹饪技巧</summary>

| 输入 | 输出 |
|------|------|
| `没有老抽可以用什么代替？` | 替代方案和建议 |
| `炒牛肉怎么才能嫩？` | 嫩肉技巧（腌制、火候等） |
| `糖醋汁的万能比例是什么？` | 经典糖醋比例和调配方法 |
</details>

## 📷 photography-mentor | 摄影导师

全面的摄影教学与相机操作指南，覆盖 12+ 主流相机品牌。

**支持品牌**：Canon、Sony、Nikon、Fujifilm、Panasonic、OM System、Leica、Hasselblad、Pentax、Ricoh、GoPro、DJI、Insta360

**功能**：
- 双路径学习：入门路径（曝光三角、构图、用光）和进阶路径（高级曝光、景深控制、创意构图）
- 基于继承架构的知识库，按品牌通用 → 具体型号逐层细化
- 支持上传相机说明书（PDF/DOC）自动解析生成知识卡片
- 中英双语摄影理论参考
